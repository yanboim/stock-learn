## 案例背景

以下是一个**模拟的波段交易完整案例**，用虚构但合理的数据演示从选股到出场的全过程。

> ⚠️ **声明：** 本案例中的公司名称、股价、财务数据均为虚构，仅用于教学目的。不构成任何投资建议。

### 交易标的：某新能源零部件公司

- **行业：** 新能源汽车零部件
- **主营业务：** 电池热管理系统
- **市值：** 约200亿
- **特点：** 细分领域龙头，客户覆盖主流新能源车企

## 第一阶段：选股（3月1日）

### 为什么关注这个行业？

1. **行业景气度高** — 新能源车渗透率持续提升
2. **政策支持** — 国家持续出台新能源相关支持政策
3. **业绩增长确定** — 头部车企销量持续增长

### 为什么关注这家公司？

1. **产业链关键环节** — 热管理系统是电池安全的核心零部件，客户绕不开
2. **客户粘性强** — 需要长期认证，一旦进入供应链很难被替换
3. **市场份额领先** — 国内市占率约25%
4. **财务数据健康** — ROE 18%，毛利率35%，负债率40%

### 选股评分

| 维度 | 评分 | 理由 |
|---|---|---|
| 行业景气度 | 8/10 | 新能源渗透率持续提升 |
| 产业链地位 | 8/10 | 卡住关键环节，不可替代性强 |
| 财务质量 | 7/10 | ROE高，现金流好，负债合理 |
| 技术面 | 6/10 | 需要等合适的买入时机 |
| 综合评分 | 7.3/10 | 纳入观察名单 |

## 第二阶段：等待买入信号（3月1日-3月15日）

### 技术面观察

3月1日到3月15日期间，股价从28元调整到25元附近：

**日线级别：**
- MA5 < MA10 < MA20（短期空头排列）
- MACD在零轴以下，DIF和DEA向下
- 成交量持续萎缩

**周线级别：**
- 股价在MA20附近（中期支撑）
- MACD在零轴以上（中期趋势仍向上）
- 前期低点24.5元附近有支撑

**判断：** 中期趋势向上，短期正在回调。等待止跌信号。

### 买入条件设定

我的交易系统要求以下条件同时满足才能买入：
1. ✅ 日线MACD金叉
2. ✅ 股价站上MA20
3. ✅ 成交量放大（超过20日均量1.5倍）
4. ✅ 出现止跌K线（锤子线/早晨之星/大阳线）

## 第三阶段：买入（3月18日）

### 买入信号出现

3月18日，出现以下信号：

1. **日线MACD金叉** — DIF上穿DEA
2. **股价突破MA20** — 收盘价25.80元，站上MA20（25.50元）
3. **成交量放大** — 成交量180万股，是20日均量（100万股）的1.8倍
4. **K线形态** — 前一天出现锤子线，今天收大阳线

### 交易计划

| 项目 | 计划 |
|---|---|
| **买入价** | 25.80元 |
| **买入仓位** | 15%（1.5万/10万总资金） |
| **止损价** | 24.00元（约-7%） |
| **目标价1** | 28.50元（+10.5%） |
| **目标价2** | 30.00元（+16.3%） |
| **风险收益比** | 1:1.5 ~ 1:2.3 |
| **最大亏损** | 1,050元（总资金的1.05%） |

### 买入记录

```
日期：3月18日
操作：买入
价格：25.80元
数量：500股
金额：12,900元 + 手续费13元 = 12,913元
仓位：约13%
理由：MACD金叉+站上MA20+放量+锤子线确认
情绪：平静（按计划执行）
```

## 第四阶段：持有与跟踪（3月18日-4月15日）

### 持有期间的波动

| 日期 | 价格 | 事件 | 操作 |
|---|---|---|---|
| 3月22日 | 26.50 | 站稳MA20，继续上涨 | 持有 |
| 3月28日 | 24.80 | 大盘下跌，股价回调到MA20附近 | 观察（未破止损） |
| 4月2日 | 26.20 | 反弹回到MA20以上 | 持有 |
| 4月8日 | 28.30 | 接近目标价1（28.50） | 准备减仓 |
| 4月10日 | 28.80 | 突破目标价1 | 卖出50% |

### 4月10日：第一次减仓

**卖出理由：** 达到目标价1（28.50以上）

```
卖出：250股 @ 28.80元
收入：7,200元 - 手续费8元 = 7,192元
剩余持仓：250股（成本25.80元）
锁定利润：7,192 - (12,913 × 50%) = 735元

止损上移到：26.50元（MA20位置）
```

## 第五阶段：出场（4月22日）

### 卖出信号出现

4月22日，出现以下信号：

1. **MACD顶背离** — 价格创新高（30.20元），但MACD没有创新高
2. **放量滞涨** — 成交量放大但价格涨不动
3. **上影线增多** — 连续两天出现上影线

### 卖出记录

```
日期：4月22日
操作：全部卖出
价格：29.60元
数量：250股
金额：7,400元 - 手续费9元 = 7,391元
理由：MACD顶背离+放量滞涨+上影线
```

## 第六阶段：交易总结

### 盈亏统计

```
买入总成本：12,913元
第一次卖出收入：7,192元
第二次卖出收入：7,391元
总收入：14,583元
净利润：14,583 - 12,913 = 1,670元
收益率：+12.9%
持有天数：35天
最大浮亏：-3.9%（3月28日）
最大浮盈：+17.1%（4月18日）
```

### 交易评分

| 维度 | 评分 | 评价 |
|---|---|---|
| **选股** | 8/10 | 行业和公司选择正确 |
| **买入时机** | 9/10 | 信号明确，入场精准 |
| **仓位管理** | 7/10 | 首次仓位偏保守，可以稍多 |
| **止损管理** | 9/10 | 止损位合理，持有期间未触发 |
| **卖出执行** | 8/10 | 分批卖出策略正确，第二次可以更早 |

### 经验总结

**做得好的：**
1. 耐心等待买入信号，没有提前入场
2. 分批卖出，既锁定了利润又留了上行空间
3. 止损位设置合理，持有期间心态稳定

**可以改进的：**
1. 首次仓位可以加到20%（信号很强时可以适当放大）
2. 第一次卖出后没有及时上移止损到成本价（如果之后大跌可能回吐利润）
3. 第二次卖出可以更早（顶背离出现当天就卖，而不是等了一天）

## 波段交易流程总结

```
1. 选股（行业+公司+基本面）
   ↓
2. 等待信号（耐心是最大的武器）
   ↓
3. 制定计划（入场价、止损、目标、仓位）
   ↓
4. 执行买入（严格按计划）
   ↓
5. 跟踪持有（移动止损、分批止盈）
   ↓
6. 卖出出场（信号确认、果断执行）
   ↓
7. 复盘总结（记录、分析、优化）
```

## 波段交易检查清单

1. ✅ 选股有明确的行业和基本面逻辑
2. ✅ 买入前设好止损、目标和仓位
3. ✅ 至少3个买入信号同时出现才入场
4. ✅ 分批买入/分批卖出降低风险
5. ✅ 盈利后及时上移止损
6. ✅ 出现卖出信号果断出场
7. ✅ 每笔交易都要复盘
