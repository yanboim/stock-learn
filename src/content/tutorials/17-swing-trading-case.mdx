## 波段交易完整流程

波段交易是指在2周到3个月内捕捉一段趋势性上涨的交易方式。下面用一个完整的模拟案例，带你走一遍从选股到出场的全流程。

:::note[声明]
本案例为教学目的构建的模拟场景，不构成任何投资建议。具体投资决策请基于最新数据独立判断。
:::

## 第一步：发现机会（选股）

### 场景设定

假设现在是某个时间点，你在做每周的选股扫描。

**选股扫描条件：**
1. 站上MA60（中线趋势向上）
2. MACD在零轴附近或上方
3. 成交量近期有所放大
4. ROE > 15%，PE < 行业平均
5. 近期有催化剂（财报、政策、行业事件等）

### 筛选结果示例

假设扫描后发现某消费类股票（以下称"示例A"）：
- 当前价格：约25元
- 市值：约200亿
- PE(TTM)：约18倍（行业平均22倍）
- ROE：约22%
- 营收增长：约15%
- 毛利率：约45%

**初步判断：**
- ✅ 基本面良好（ROE高、PE合理偏低）
- ✅ 有估值修复空间（PE低于行业平均）
- ✅ 营收在增长
- 🔍 需要进一步技术面分析

## 第二步：技术分析（确认）

### 大趋势判断

**周线级别：**
- 价格在MA60上方 → 中长期趋势向上 ✅
- MACD在零轴上方 → 多头趋势 ✅
- 均线MA20 > MA60 → 中期趋势向上 ✅

**结论：大趋势向上，可以做多。**

### 入场时机分析

**日线级别：**
- MACD从零轴下方开始金叉 → 买入信号出现 ✅
- 价格刚好站上MA20 → 短期趋势确认 ✅
- 成交量放大至5日均量1.5倍 → 资金确认 ✅
- RSI从35回升到55 → 从超卖区回升 ✅

**结论：多个信号共振，入场时机较好。**

### 支撑压力分析

- 下方支撑：MA20（约24元）、前期低点（约23元）
- 上方压力：前高（约28元）、整数关口30元

## 第三步：制定交易计划

### 计划内容

**买入计划：**
- 入场条件：已满足（MACD金叉 + 站上MA20 + 放量）
- 买入价位：25-25.5元区间
- 分批买入：
  - 第一批（50%仓位）：25元附近
  - 第二批（50%仓位）：确认突破25.5元后

**止损计划：**
- 固定止损：跌破23.5元（-6%）→ 清仓
- 均线止损：收盘连续2天在MA20以下 → 减仓50%
- 逻辑止损：季报业绩大幅低于预期 → 清仓

**止盈计划：**
- 第一目标：28元（+12%）→ 减仓50%
- 第二目标：30元（+20%）→ 再减30%
- 剩余仓位用移动止损（回撤8%就卖）

**仓位管理：**
- 单只股票占总资金15%
- 分2批建仓
- 总仓位不超过70%

### 风险收益比评估

```
预期收益：(28-25) / 25 = 12%（保守目标）
预期风险：(25-23.5) / 25 = 6%（止损幅度）
风险收益比 = 12% / 6% = 2:1 ✅

如果看到30元：
风险收益比 = 20% / 6% = 3.3:1 ✅✅
```

**风险收益比 > 2:1，值得操作。**

## 第四步：执行交易

### 第一批买入

- 价格在25.1元时买入50%仓位
- 同时设置条件止损单：23.5元卖出
- 记录交易日志

### 第二批加仓

- 3天后价格突破25.8元，成交量继续放大
- 在25.8元加仓剩余50%
- 调整止损到24.5元（保住部分本金）

## 第五步：持仓管理

### 第1周

- 价格在25.5-26.5之间波动
- MACD红柱持续 → 趋势健康
- 继续持有，不做任何操作

### 第2周

- 价格突破27元，接近第一目标
- 成交量放大 → 突破有效
- 在27.5元减仓50%（部分止盈）
- 剩余仓位移动止损上移到26元

### 第3周

- 价格继续上涨到28.5元
- MACD红柱开始缩短 → 动量减弱
- 在28.5元再减30%（接近第二目标）
- 剩余20%仓位移动止损上移到27元

### 第4周

- 价格回调到27.8元
- 触发移动止损 → 在27元卖出剩余20%

## 第六步：交易复盘

### 交易总结

| 项目 | 内容 |
|---|---|
| 买入价 | 第一批25.1，第二批25.8 |
| 卖出价 | 27.5（50%）、28.5（30%）、27（20%） |
| 持仓天数 | 约4周 |
| 综合收益率 | 约+9.5% |
| 最大回撤 | 持仓期间约-3% |

### 做对了什么

- ✅ 按计划分批买入和卖出
- ✅ 严格执行止损和移动止损
- ✅ 没有因为情绪改变计划
- ✅ 风险收益比控制得好

### 可以改进

- 🔧 第一批买入可以再等等，等回调到MA20附近再进
- 🔧 减仓可以更积极一些，MACD红柱缩短时就应该减
- 🔧 应该更关注行业新闻，及时了解催化剂变化

## 波段交易的核心要点

1. **趋势为王** — 只在上升趋势中做多
2. **信号共振** — 多个指标同时确认，可靠性更高
3. **分批操作** — 买入和卖出都分批进行
4. **严格止损** — 到了止损位无脑执行
5. **移动止盈** — 用移动止损保护已有利润
6. **保持纪律** — 按计划执行，不因情绪改变
7. **及时复盘** — 每笔交易结束后总结经验教训

## 常见问题

**Q：如果买入后立刻下跌怎么办？**
A：到了止损位就执行，不要犹豫。止损是交易的成本，就像开店的租金一样正常。

**Q：如果减仓后继续大涨怎么办？**
A：你已经赚了你计划赚的钱，不要后悔。市场永远有赚不完的钱，但亏起来可以很快。

**Q：一次波段交易应该赚多少？**
A：根据风险收益比来定，一般目标10-20%。不要贪心，赚自己认知范围内的钱。
