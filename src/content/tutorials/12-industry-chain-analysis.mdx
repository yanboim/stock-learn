## 为什么要做产业链分析？

选股票就像选供应商——你要找的不是"看起来不错"的公司，而是**整个产业链条上最不可替代的那一环**。

:::tip[核心思路]
最好的投资标的通常具备以下特征：
1. 卡住产业链的**关键瓶颈**
2. 下游客户**绕不开**它
3. 替代成本非常高
4. 议价能力强（能吃到行业增长的大部分利润）
:::

## 产业链分析框架

### 任何行业都可以简化为三层

```
┌─────────────┐
│   上游       │  原材料 / 基础设备 / 核心技术
│  （供应商）   │
├─────────────┤
│   中游       │  制造 / 加工 / 组装 / 集成
│  （生产商）   │
├─────────────┤
│   下游       │  终端产品 / 品牌 / 渠道 / 用户
│  （客户）    │
└─────────────┘
```

### 关键问题：钱被谁赚走了？

产业链中利润分配是不均匀的。**谁掌握了关键环节，谁就分走了最大的利润。**

> 例子：智能手机产业链
> - 上游：高通芯片（利润率极高，不可替代）
> - 中游：富士康代工（利润率很低，可替代性强）
> - 下游：苹果品牌（利润率最高，掌握用户和生态）
>
> 钱被高通和苹果赚走了，富士康只赚辛苦钱。

## 五步产业链分析法

### 第一步：画出产业链地图

列出行业从原材料到终端用户的所有环节：

> 例：新能源汽车产业链
> ```
> 锂矿开采 → 正极材料 → 电池制造 → 电机电控 → 整车组装 → 销售渠道 → 终端用户
>         → 负极材料 ↗
>         → 电解液  ↗
>         → 隔膜    ↗
> ```

### 第二步：找到关键瓶颈

问：**哪个环节的产能最紧张 / 技术壁垒最高 / 供应最不可替代？**

> 新能源汽车的关键瓶颈：
> - 动力电池（宁德时代、比亚迪）— 技术壁垒高，认证周期长
> - 车规级芯片 — 供给有限，认证严格
> - 上游锂矿 — 资源稀缺，扩产周期长

### 第三步：评估议价能力

用五个维度评估每个环节的议价能力：

**1. 客户集中度**
- 只有1-2个大客户 → 议价能力弱（客户说了算）
- 客户众多且分散 → 议价能力强（不依赖单一客户）

**2. 替代难度**
- 客户换供应商成本很低 → 议价能力弱
- 客户换供应商需要重新认证（半年以上）→ 议价能力强

**3. 技术壁垒**
- 技术成熟，门槛低 → 容易被替代
- 技术专利多，研发投入大 → 不容易被替代

**4. 产能稀缺性**
- 产能充足甚至过剩 → 价格战，议价能力弱
- 产能紧张，供不应求 → 涨价能力强

**5. 品牌和生态**
- 无品牌，同质化 → 只能拼价格
- 强品牌或生态壁垒 → 客户黏性强

### 第四步：分析利润分配

看每个环节的**毛利率**和**净利率**：

| 环节 | 典型毛利率 | 利润水平 |
|---|---|---|
| 核心技术/专利 | 50-80% | 💰💰💰💰💰 |
| 核心零部件 | 30-50% | 💰💰💰💰 |
| 组装/代工 | 5-15% | 💰 |
| 品牌/渠道 | 40-70% | 💰💰💰💰💰 |

**利润率最高的环节，通常就是产业链中最有价值的环节。**

### 第五步：判断未来变化

产业链的价值分配不是一成不变的：

- 技术变革可能导致价值转移（如燃油车→电动车）
- 供需关系变化影响议价能力（如锂矿从紧缺到过剩）
- 政策变化影响利润分配（如补贴退坡）

## 实战案例：半导体产业链分析

### 产业链地图

```
EDA软件 → 芯片设计 → 晶圆制造 → 封装测试 → 终端应用
  (上游)    (上游)     (中游)     (中游)     (下游)
```

### 关键瓶颈

| 环节 | 代表公司 | 壁垒 | 议价能力 |
|---|---|---|---|
| EDA软件 | Synopsys、Cadence | ⭐⭐⭐⭐⭐ | 极强（全球寡头垄断） |
| 光刻机 | ASML | ⭐⭐⭐⭐⭐ | 极强（全球唯一高端供应商） |
| 晶圆代工 | 台积电、中芯国际 | ⭐⭐⭐⭐⭐ | 极强（技术和规模壁垒） |
| 芯片设计 | 英伟达、AMD、海思 | ⭐⭐⭐⭐ | 强（技术驱动） |
| 封装测试 | 长电科技等 | ⭐⭐ | 一般（技术壁垒相对较低） |

### 投资启示

- **最不可替代**：光刻机（ASML）、EDA软件、先进制程代工
- **高增长弹性**：AI芯片设计（英伟达等）
- **国产替代空间大**：半导体设备、材料、EDA（中国公司机会多）
- **相对可替代**：传统封装测试（竞争激烈）

## 中国优势产业链

### 全球领先

- **光伏**：硅料→硅片→电池片→组件，中国占全球70%+份额
- **锂电池**：宁德时代+比亚迪，全球动力电池份额超60%
- **稀土加工**：全球90%稀土加工在中国
- **新能源整车**：比亚迪全球销量领先

### 正在追赶

- **半导体**：中芯国际（制造）、北方华创（设备）、华大九天（EDA）
- **航空航天**：C919大飞机、商业航天
- **高端机床**：数控系统、精密加工

### 投资产业链核心公司的要点

1. **确认关键瓶颈** — 它是否卡住了行业的关键节点？
2. **评估不可替代性** — 客户能绕开它吗？替代者是谁？
3. **看毛利率趋势** — 毛利率在提升=议价能力在增强
4. **看研发投入** — 持续高研发=在巩固壁垒
5. **关注行业增长** — 好公司也要在增长的行业里

## 常见错误

### ❌ 只看公司不看产业链

一家公司财务数据再好，如果处于产业链中可替代的环节，长期竞争力堪忧。

### ❌ 认为技术最先进就是最好的投资

技术先进 ≠ 商业成功。还要看成本、客户、生态和商业模式。

### ❌ 忽略产业链变化

行业技术路线转变时，原来的龙头可能被颠覆。
