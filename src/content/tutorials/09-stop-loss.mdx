## 为什么止损比止盈更重要？

先看一个残酷的数学事实：

| 亏损幅度 | 需要涨多少才能回本 |
|---|---|
| 亏损10% | 需要涨 **11%** |
| 亏损20% | 需要涨 **25%** |
| 亏损30% | 需要涨 **43%** |
| 亏损50% | 需要涨 **100%** |
| 亏损70% | 需要涨 **233%** |
| 亏损90% | 需要涨 **900%** |

**亏损是复利的敌人。亏50%需要翻倍才能回来，但翻倍有多难？**

:::tip[第一条铁律]
**活下来比赚多少钱更重要。**

止损不是为了"认输"，而是为了"保命"。
就像开车系安全带——不是为了计划出车祸，而是万一出事时保护自己。
:::

## 止损的心理障碍

### "我再等等，说不定会涨回来"

这是最危险的想法。很多从亏损10%扛到亏损70%的悲剧，都是这个心态。

### "已经亏这么多了，现在卖太亏了"

**沉没成本谬误**。你买入的价格不重要，重要的是这只股票现在值不值得继续持有。

### "一卖就涨怎么办？"

确实可能发生。但你要问自己：**如果这次不止损，会不会养成不止损的习惯？**

偶尔止损错了没关系，养成了不止损的习惯才会致命。

## 7种实战止损方法

### 方法一：固定比例止损

**规则**：买入后亏损达到X%就卖出

| 风格 | 建议止损比例 |
|---|---|
| 短线 | 3-5% |
| 波段 | 5-8% |
| 中线 | 8-12% |
| 长线 | 15-20% |

**优点**：简单明确，容易执行
**缺点**：没有考虑股票的波动特性

### 方法二：关键均线止损

**规则**：跌破某条关键均线就卖出

- 短线：跌破MA5或MA10
- 波段：跌破MA20
- 中线：跌破MA60
- 长线：跌破MA120

**优点**：考虑了趋势变化
**缺点**：有时跌破均线后又会涨回来（假跌破）

:::tip[增强版]
**收盘跌破均线**才算有效跌破，盘中跌破不算。连续2天收盘在均线下方才止损。
:::

### 方法三：前低止损

**规则**：跌破最近的支撑低点就卖出

- 买入时找到最近的一个明显低点
- 如果价格跌破这个低点 → 止损

**优点**：基于真实的价格支撑位
**缺点**：支撑位有时会被假跌破

### 方法四：技术形态止损

**规则**：技术形态被破坏就卖出

- 突破买入后，价格回到突破位以下
- 底部形态（如双底）的颈线被跌破
- 趋势线被跌破

### 方法五：逻辑止损（最高级）

**规则**：买入逻辑被证伪就卖出

> 例子：你因为公司业绩大增而买入 → 结果发现业绩造假 → 买入逻辑完全失效 → 立刻止损

> 例子：你因为行业政策利好买入 → 结果政策被取消 → 买入逻辑不再成立 → 止损

**这是最根本的止损方法——问问自己当初为什么买，这个理由还在不在？**

### 方法六：时间止损

**规则**：买入后X天/周没有按预期方向走 → 止损

- 波段交易：买入后2周没涨 → 止损
- 中线投资：买入后1个月没涨 → 止损

**核心理念**：时间也是成本，资金占用在不动的地方就错过了其他机会。

### 方法七：移动止损（跟踪止盈）

**规则**：随着价格上涨，不断上移止损位

```
买入价10元，止损设9元（-10%）
涨到12元 → 止损上移到11元（保住1元利润）
涨到14元 → 止损上移到12.5元（保住2.5元利润）
回调到12.5元 → 触发止损，但已经赚了25%
```

**优点**：既能保护利润，又能让利润奔跑
**推荐**：涨幅超过10%后开始使用移动止损

## 止损的黄金法则

### 法则一：买入前就设好止损

不要等亏了才想"要不要卖"。**在买入的那一刻就决定好在什么价位止损。**

### 法则二：止损位一旦设定不随意下移

止损可以上移（保护利润），但绝不能下移（扩大亏损）。

### 法则三：到了就执行，不犹豫

到了止损位就像红灯一样——你不会在红灯时犹豫要不要停。

### 法则四：止损后不马上买回

止损后至少等1-2天再考虑重新入场，避免情绪化操作。

### 法则五：记录每一笔止损

记录：买了什么、为什么买、为什么止损、止损后股价走势。长期来看你会发现止损决策中大部分是对的。

## 止损之外的仓位保护

止损是**单笔保护**，还需要**整体保护**：

- 总仓位止损：当月总亏损达到8% → 降低仓位至50%以下
- 连续止损保护：连续3笔止损 → 暂停交易，复盘总结
- 黑天鹅保护：永远不满仓，至少保留20%现金
