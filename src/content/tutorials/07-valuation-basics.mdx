## 为什么要学估值？

买股票最怕什么？**买贵了。**

一家好公司，如果你买入的价格太高，也可能亏钱。一家一般的公司，如果你买入的价格够便宜，也可能赚钱。

估值就是帮你看清：**这家公司现在到底是贵还是便宜。**

:::tip[核心思想]
估值不是预测股价涨跌，而是判断当前价格是否合理。
就像买房——你需要知道这个地段的房子值多少钱，才能判断房东开的价格是高还是低。
:::

## PE（市盈率）：几年回本？

### 通俗理解

PE = 股价 ÷ 每股收益 = 买入这家公司后，靠利润**几年能回本**

> 一家奶茶店每年赚10万，你花100万买下来 → PE = 100÷10 = **10倍**
> 含义：按当前利润水平，10年回本。

### 计算公式

```
PE = 股价 / 每股收益(EPS)
或
PE = 总市值 / 净利润
```

### PE的高低判断

| PE范围 | 含义 | 类比 |
|---|---|---|
| PE < 10 | 很便宜（或公司有问题） | 奶茶店3年回本 |
| PE 10-20 | 合理偏低 | 奶茶店5-10年回本 |
| PE 20-40 | 合理偏高 | 奶茶店10-20年回本 |
| PE > 40 | 昂贵（或高增长预期） | 奶茶店20年以上回本 |
| PE < 0 | 公司亏损！ | 奶茶店年年亏钱 |

:::caution[注意]
PE不是越低越好！
- PE很低可能是因为公司业绩在下滑，市场不看好
- PE很高可能是因为公司高速增长，市场愿意给溢价
- **一定要同行业对比**，不同行业的PE水平完全不同
:::

### 静态PE vs 动态PE vs 滚动PE

| 类型 | 计算 | 特点 |
|---|---|---|
| **静态PE（LYR）** | 用去年全年利润 | 最准确但滞后 |
| **滚动PE（TTM）** | 用最近4个季度利润 | 最常用，兼顾时效和准确 |
| **动态PE（Forward）** | 用今年预期利润 | 前瞻性但预期可能不准 |

**建议主要看滚动PE（TTM）**

## PB（市净率）：打折还是溢价？

### 通俗理解

PB = 股价 ÷ 每股净资产 = 你花多少钱去买**1块钱的资产**

> 一家公司的净资产（总资产-总负债）是1亿，市值是2亿 → PB = 2倍
> 含义：你花2块钱去买公司1块钱的资产。

### 计算公式

```
PB = 股价 / 每股净资产
或
PB = 总市值 / 净资产
```

### PB的高低判断

| PB范围 | 含义 |
|---|---|
| **PB < 1** | 破净——市值低于净资产，相当于打折买资产 |
| PB 1-3 | 合理范围 |
| PB > 5 | 溢价较高，市场对公司未来非常看好 |
| PB > 10 | 极高溢价，通常出现在科技/互联网公司 |

:::tip[什么时候看PB？]
- **重资产行业**（银行、钢铁、煤炭、房地产）→ 看PB更合适
- **轻资产行业**（软件、互联网、咨询）→ 看PE更合适
- **亏损公司**（PE为负没有意义）→ 看PB
:::

## ROE（净资产收益率）：赚钱效率

### 通俗理解

ROE = 公司用股东的钱能赚多少利润 = **赚钱效率**

> 你投资100万开奶茶店，一年赚了20万 → ROE = 20%
> 隔壁老王也投100万开奶茶店，一年赚了5万 → ROE = 5%
> 你比老王的赚钱效率高4倍！

### 计算公式

```
ROE = 净利润 / 净资产 × 100%
```

### ROE的高低判断

| ROE范围 | 评价 |
|---|---|
| **ROE > 20%** | 优秀！巴菲特最喜欢的标准 |
| ROE 15-20% | 良好，优质公司 |
| ROE 10-15% | 及格，一般公司 |
| ROE 5-10% | 较差，赚钱效率低 |
| ROE < 5% | 差，不如把钱存银行 |

:::tip[巴菲特的名言]
"如果只能用一个指标来选股，我会选择ROE。"
——沃伦·巴菲特

**持续高ROE的公司**（连续5年以上ROE > 15%）往往具有强大的竞争优势。
:::

## 三个指标怎么组合使用？

### 组合一：高ROE + 低PE = 黄金标的

- ROE高 → 赚钱效率高
- PE低 → 价格便宜
- **这就是价值投资梦寐以求的标的！**

但要注意：市场不是傻子，这种机会通常只在以下情况出现：
- 市场恐慌性下跌（整体熊市）
- 公司暂时遇到困难但长期竞争力仍在
- 市场对公司有误解或偏见

### 组合二：高ROE + 高PE = 成长股

- ROE高 → 赚钱效率高
- PE高 → 价格不便宜
- 市场在为未来的高增长买单

如果你相信公司能持续高增长，可以接受较高的PE。
但风险是：一旦增长不及预期，股价可能大幅回调。

### 组合三：低ROE + 低PE = 价值陷阱

- ROE低 → 赚钱效率低
- PE低 → 看起来便宜
- ⚠️ **便宜有便宜的道理！** 可能是公司竞争力持续下滑

**不能只看PE低就买！** 一定要看ROE确认公司赚钱能力。

### 组合四：低ROE + 高PE = 坚决回避

既不赚钱又贵——完全不符合投资逻辑。

## 不同行业的估值标准

| 行业 | 参考PE | 参考PB | 关键看什么 |
|---|---|---|---|
| 银行 | 5-8倍 | 0.5-1倍 | PB更关键，ROE稳定性 |
| 白酒/消费 | 20-40倍 | 5-15倍 | 品牌力、ROE稳定性 |
| 科技/半导体 | 30-80倍 | 3-10倍 | 增长率、研发投入 |
| 医药 | 20-50倍 | 3-8倍 | 研发管线、政策影响 |
| 新能源 | 20-60倍 | 2-6倍 | 产能、行业景气度 |
| 房地产 | 5-10倍 | 0.3-1倍 | PB更关键，负债风险 |

:::note[核心原则]
**永远不要跨行业直接比较PE！** 科技股PE 30倍可能很便宜，银行股PE 30倍就贵得离谱。
同行业内对比才有意义。
:::

## 实战检查清单

1. ✅ 先看ROE — 公司赚钱效率高不高
2. ✅ 再看PE/PB — 当前价格贵不贵
3. ✅ 同行业对比 — 相对估值水平如何
4. ✅ 看历史区间 — 当前估值在历史中处于什么位置
5. ✅ 理解贵的原因 — 高PE是因为高增长还是泡沫？
