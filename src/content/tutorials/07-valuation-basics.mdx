## 为什么要学估值？

买股票最怕什么？**买贵了。**

一家好公司，如果你买入的价格太高，也可能亏钱。一家一般的公司，如果你买入的价格够便宜，也可能赚钱。

估值就是帮你看清：**这家公司现在到底是贵还是便宜。**

## PE（市盈率）：几年回本？

**PE = 股价 ÷ 每股收益 = 市值 ÷ 净利润**

通俗理解：**你花多少年能通过公司利润收回投资。**

<div class="visual-card" style="background:#eff6ff;border-color:#bfdbfe">
  <strong style="color:#1e40af">🧮 PE 计算示例</strong>
  <div style="margin-top:0.75rem;padding:1rem;background:white;border-radius:0.5rem">
    <p style="margin:0;color:#475569">某公司市值 100亿，去年净利润 5亿</p>
    <p style="margin:0.5rem 0 0;font-size:1.2rem;font-weight:bold;color:#1e40af">PE = 100 ÷ 5 = 20倍</p>
    <p style="margin:0.5rem 0 0;color:#6b7280;font-size:0.9rem">意思是：按现在的利润水平，需要20年回本</p>
  </div>
</div>

:::caution[PE 陷阱]
PE低不一定就是便宜！可能是：
- 公司利润在下滑（PE低因为E在增长但P没跟上）
- 周期性行业在景气高点（利润虚高导致PE虚低）
- 公司有一次性收益（利润虚高）
:::

## PB（市净率）：溢价多少？

**PB = 股价 ÷ 每股净资产 = 市值 ÷ 净资产**

通俗理解：**你花了多少倍的价钱去买公司的"家底"。**

<div class="visual-card" style="background:#ede9fe;border-color:#c4b5fd">
  <strong style="color:#5b21b6">🧮 PB 计算示例</strong>
  <div style="margin-top:0.75rem;padding:1rem;background:white;border-radius:0.5rem">
    <p style="margin:0;color:#475569">某公司市值 100亿，净资产 50亿</p>
    <p style="margin:0.5rem 0 0;font-size:1.2rem;font-weight:bold;color:#5b21b6">PB = 100 ÷ 50 = 2倍</p>
    <p style="margin:0.5rem 0 0;color:#6b7280;font-size:0.9rem">意思是：你花2元买公司1元的净资产</p>
  </div>
</div>

## ROE（净资产收益率）：赚钱效率

**ROE = 净利润 ÷ 净资产 × 100%**

通俗理解：**公司用股东的钱，每年能赚回百分之多少。**

<div class="visual-card" style="background:#f0fdf4;border-color:#bbf7d0">
  <strong style="color:#065f46">🧮 ROE 计算示例</strong>
  <div style="margin-top:0.75rem;padding:1rem;background:white;border-radius:0.5rem">
    <p style="margin:0;color:#475569">某公司净利润 10亿，净资产 50亿</p>
    <p style="margin:0.5rem 0 0;font-size:1.2rem;font-weight:bold;color:#047857">ROE = 10 ÷ 50 × 100% = 20%</p>
    <p style="margin:0.5rem 0 0;color:#6b7280;font-size:0.9rem">意思是：每100元股东权益，一年赚20元</p>
  </div>
</div>

## 杜邦分析：ROE的拆解

ROE看起来很高，但你需要知道它为什么高——是用杠杆（借钱）堆出来的，还是靠真正的经营效率？

**杜邦公式：**
```
ROE = 净利率 × 周转率 × 权益乘数
```

## PEG：考虑增长的估值

PE只看当前利润，不考虑增长。PEG加入了增长因素：

**PEG = PE ÷ 净利润增长率**

## 快速估值筛选模板

```
公司：___________
当前PE：_______（同行业平均：_______）
当前PB：_______（同行业平均：_______）
ROE：_______%（连续3年 &gt; 15%？✅/❌）
PEG：_______（&lt; 1？✅/❌）

PE历史分位：_______%（&lt; 30% 可能低估）
同行业对比：PE高于/低于行业平均

估值结论：低估 / 合理 / 高估
```
