## 为什么要看财务报表？

财务报表是公司的"体检报告"。通过财务报表，你可以看出：

- **这家公司赚不赚钱？**（利润表）
- **这家公司有多少家底？**（资产负债表）
- **这家公司的钱从哪来到哪去？**（现金流量表）

:::tip[核心思想]
如果你买了一家公司的股票，你就有权利知道这家公司的财务状况。财务报表就是公司定期向你汇报的"成绩单"。
:::

## 三大报表速览

### 利润表（损益表）— 赚不赚钱？

利润表告诉你公司在一段时间内赚了多少钱。

**简化版：**
```
营业收入（卖了多少钱）
  - 营业成本（进货/生产花了多少）
  = 毛利润
  - 销售费用（广告、销售人员工资）
  - 管理费用（管理人员工资、办公费）
  - 研发费用（技术研究投入）
  - 财务费用（利息支出）
  = 营业利润
  + 营业外收支（非经常性损益）
  - 所得税
  = 净利润（最终赚了多少钱）
```

**关键指标：**

| 指标 | 计算公式 | 健康标准 |
|---|---|---|
| **毛利率** | 毛利润/营业收入 | > 30%（越高越好） |
| **净利率** | 净利润/营业收入 | > 10%（优秀公司 > 20%） |
| **营收增速** | (本期-上期)/上期 | > 10%（稳定增长） |
| **净利增速** | (本期-上期)/上期 | > 营收增速（经营杠杆） |

**利润表的红灯信号：**
- ⚠️ 营收增长但净利润下降（增收不增利）
- ⚠️ 毛利率持续下降（竞争加剧或成本上升）
- ⚠️ 非经常性损益占比过高（靠变卖资产/政府补贴维持利润）
- ⚠️ 应收账款增速远超营收增速（可能虚增收入）

### 资产负债表 — 有多少家底？

资产负债表告诉你公司在某一天"有什么"和"欠什么"。

**核心公式：**
```
资产 = 负债 + 所有者权益

（我有什么）= （我欠别人的）+ （真正属于自己的）
```

**资产端（有什么）：**

| 资产类型 | 说明 | 关注点 |
|---|---|---|
| **货币资金** | 现金和银行存款 | 越多越好（偿债能力强） |
| **应收账款** | 客户欠你的钱 | 占营收比例越低越好 |
| **存货** | 仓库里的商品 | 周转越快越好 |
| **固定资产** | 厂房、设备 | 折旧是否合理 |
| **无形资产** | 专利、品牌、商誉 | 商誉过高要警惕 |

**负债端（欠什么）：**

| 负债类型 | 说明 | 关注点 |
|---|---|---|
| **短期借款** | 一年内要还的钱 | 能否按时偿还 |
| **应付账款** | 欠供应商的钱 | 适度说明议价能力强 |
| **长期借款** | 一年以后还的钱 | 利率和用途 |
| **预收账款** | 客户提前付的钱 | 越多越好（说明产品抢手） |

**关键指标：**

| 指标 | 计算公式 | 健康标准 |
|---|---|---|
| **资产负债率** | 总负债/总资产 | < 60%（银行地产除外） |
| **流动比率** | 流动资产/流动负债 | > 1.5 |
| **速动比率** | (流动资产-存货)/流动负债 | > 1.0 |
| **应收账款周转天数** | 365/周转率 | 越短越好 |

### 现金流量表 — 钱从哪来？

现金流量表告诉你公司的现金是怎么流进流出的。

**三大现金流：**

| 类型 | 含义 | 健康标准 |
|---|---|---|
| **经营现金流** | 日常经营赚到的现金 | 必须为正，最好 > 净利润 |
| **投资现金流** | 买卖资产/投资的现金 | 为负正常（在扩张），但不能长期大额为正（在变卖资产） |
| **筹资现金流** | 借款/还款/分红/融资 | 看具体情况 |

**现金流组合判断：**

| 经营 | 投资 | 筹资 | 公司阶段 |
|---|---|---|---|
| **正** | 负 | 正 | 成长期（赚钱+投资+借钱扩张） |
| **正** | 负 | 负 | 成熟期（赚钱+投资+还债/分红）✅ 最健康 |
| **负** | 正 | 正 | 困难期（亏钱+变卖资产+借钱度日）⚠️ |
| **负** | 负 | 正 | 烧钱期（初创公司常见）⚠️ |

## 财务造假识别

### 常见造假手法

**1. 虚增收入**
- 特征：应收账款暴增，经营现金流不匹配
- 识别：应收账款增速远超营收增速

**2. 费用资本化**
- 特征：把该计入当期费用的支出算成资产
- 识别：无形资产异常增长，研发投入资本化率过高

**3. 存货虚增**
- 特征：虚增存货价值
- 识别：存货周转天数大幅增长

**4. 关联交易**
- 特征：和关联方做不公正的交易
- 识别：关联交易占比过高，交易价格不公允

### 财务质量检查清单

1. ✅ 经营现金流 > 净利润（利润含金量高）
2. ✅ 应收账款增速 < 营收增速
3. ✅ 存货周转天数稳定或下降
4. ✅ 毛利率和净利率趋势稳定
5. ✅ 负债率合理（非金融企业 < 60%）
6. ✅ 商誉占净资产比例 < 20%
7. ✅ 审计意见为"标准无保留意见"

## 如何快速扫描一家公司的财务状况

### 5分钟快速评估

**1. 看ROE（1分钟）**
- ROE > 15% → 值得深入研究
- ROE < 10% → 除非有特殊原因，基本可以跳过

**2. 看利润表（1分钟）**
- 营收是否持续增长？
- 净利润增速是否匹配？
- 毛利率是否稳定？

**3. 看现金流（1分钟）**
- 经营现金流是否为正？
- 经营现金流是否 > 净利润？

**4. 看负债（1分钟）**
- 资产负债率是否合理？
- 有没有大额短期借款？

**5. 看异常（1分钟）**
- 有没有突然大幅波动的项目？
- 应收/存货有没有异常增长？

```
毛利率：_____%（稳定或上升? ✅/❌）
净利率：_____%（> 10%? ✅/❌）

财务质量评分：___/10
值得进一步研究？是/否
```
