## 财务报表：公司的体检报告

看财务报表就像看体检报告：
- **资产负债表** — 身体的骨骼和肌肉状况（有多少家底）
- **利润表** — 最近一年的营养摄入和消耗（赚了还是亏了）
- **现金流量表** — 血液循环系统（钱的来龙去脉）

:::tip[核心原则]
**利润可以美化，但现金很难造假。**
一家公司利润再好看，如果现金流持续为负，就像一个人收入很高但银行账户永远没钱——肯定有问题。
:::

## 资产负债表：公司的家底

### 通俗理解

资产负债表就是一个"天平"：

```
资产（你有什么）= 负债（你欠什么）+ 所有者权益（真正属于股东的）
```

> 个人类比：
> - 资产：房子（500万）+ 存款（50万）= 550万
> - 负债：房贷（300万）
> - 所有者权益：550 - 300 = 250万（你真正的净资产）

### 关键指标

**1. 资产负债率 = 总负债 ÷ 总资产**

| 资产负债率 | 评价 |
|---|---|
| < 30% | 非常安全，财务保守 |
| 30-50% | 健康，大多数好公司在这个范围 |
| 50-70% | 需要关注，部分行业正常（如银行、房地产） |
| > 70% | 风险较高，偿债压力大 |

:::note[行业差异]
- 银行资产负债率通常90%以上（正常，因为银行就是靠借钱生钱）
- 房地产60-80%（靠借钱盖楼）
- 科技/消费公司通常20-40%
:::

**2. 流动比率 = 流动资产 ÷ 流动负债**

- > 2 → 短期偿债能力好
- 1-2 → 正常
- < 1 → ⚠️ 短期可能有偿债压力

**3. 商誉占净资产比例**

- 商誉 = 收购时多花的钱（买的贵不贵）
- 商誉/净资产 > 30% → ⚠️ 商誉减值风险大

### 健康的资产负债表长什么样

- ✅ 货币资金充裕（账上有钱）
- ✅ 应收账款增长 < 营收增长（不是靠赊账做大收入）
- ✅ 存货周转正常（没有大量积压）
- ✅ 有息负债占比合理（利息负担不重）
- ✅ 资产负债率在同行业合理水平

## 利润表：赚了多少钱

### 通俗理解

利润表记录公司在一段时间内赚了多少、花了多少、最终剩多少。

```
营业收入（卖了多少钱）
 - 营业成本（进货/生产花了多少）
 = 毛利润
 - 三项费用（销售费用+管理费用+研发费用+财务费用）
 = 营业利润
 + 营业外收入 - 营业外支出
 = 利润总额
 - 所得税
 = 净利润（最终赚到口袋的钱）
```

### 关键指标

**1. 毛利率 = 毛利润 ÷ 营业收入**

| 毛利率 | 含义 |
|---|---|
| > 60% | 强品牌/高技术壁垒（如茅台90%+、游戏公司80%+） |
| 30-60% | 正常的优质公司 |
| 15-30% | 一般制造业 |
| < 15% | 苦逼行业，利润薄如纸 |

**核心逻辑：毛利率越高，产品竞争力越强，定价权越大。**

**2. 净利率 = 净利润 ÷ 营业收入**

- > 20% → 优秀
- 10-20% → 良好
- 5-10% → 一般
- < 5% → 较差

**3. 营收增长率**

- > 30% → 高速增长
- 15-30% → 稳定增长
- 5-15% → 成熟企业
- < 0% → 衰退（需要警惕）

**4. 扣非净利润**

扣除非经常性损益后的净利润 = 真正靠主营业务赚的钱

- 如果净利润高但扣非净利润低 → 公司利润不靠主业（可能靠卖资产、政府补贴）
- ⚠️ 这种利润不可持续

### 健康的利润表长什么样

- ✅ 营收持续增长
- ✅ 毛利率稳定或提升
- ✅ 费用率（三项费用/营收）稳定或下降
- ✅ 净利润增长 ≥ 营收增长（规模效应在体现）
- ✅ 扣非净利润占比高（利润来自主业）

## 现金流量表：钱的来龙去脉

### 通俗理解

现金流量表记录了公司的钱从哪来、到哪去：

- **经营现金流** — 做生意赚到的真金白银
- **投资现金流** — 买设备、建厂房、收购花了多少钱
- **筹资现金流** — 借钱、还钱、分红

:::tip[最重要的指标]
**经营现金流净额** = 公司做主业真正收到的现金

- 经营现金流 > 净利润 → 利润含金量高，赚的是真钱 ✅
- 经营现金流 < 净利润 → ⚠️ 利润可能"好看不中用"（应收账款多、没收到真金白银）
- 经营现金流长期为负 → 🚨 公司在做赔本买卖
:::

### 三大现金流的8种组合

| 经营 | 投资 | 筹资 | 公司类型 | 评价 |
|---|---|---|---|---|
| **+** | - | + | 成长期公司 | 正在扩张，需要外部融资 |
| **+** | - | - | 成熟优质公司 | **最理想！** 赚钱+投资+还债/分红 |
| **+** | + | - | 收缩期公司 | 在变卖资产，但经营还行 |
| **-** | - | + | 创业期公司 | 还没盈利，烧钱扩张中 |
| **-** | + | + | **危险公司** | 经营不赚钱，靠借钱和卖资产续命 🚨 |
| **-** | - | - | **濒危公司** | 全面出血，随时可能倒闭 🚨🚨 |

## 识别财务造假的红旗

### 🚩 红旗一：营收增长但经营现金流下降

赚的钱越来越多，但收回来的现金越来越少 → 可能在虚增收入

### 🚩 红旗二：应收账款增速远超营收增速

东西卖出去了但收不到钱 → 可能是放宽信用条件冲业绩，甚至虚假销售

### 🚩 红旗三：存货暴增

生产了很多但卖不出去 → 可能面临减值风险

### 🚩 红旗四：频繁更换审计师

好端端的为什么要换审计？→ 可能是前任审计不配合做账

### 🚩 红旗五：关联交易占比过高

跟"自己人"做生意 → 价格可能不公允，利润可能被操控

### 🚩 红旗六：利润全靠政府补贴或投资收益

主业不赚钱 → 利润不可持续

## 杜邦分析：ROE的三驾马车

ROE可以拆分为三个因子的乘积：

```
ROE = 净利率 × 资产周转率 × 权益乘数
     (赚钱能力) (运营效率) (杠杆水平)
```

| 驱动因素 | 高ROE靠什么 | 例子 |
|---|---|---|
| 高净利率 | 产品定价权强 | 茅台（净利率50%+） |
| 高周转率 | 运营效率高 | 沃尔玛（薄利多销） |
| 高杠杆 | 借钱多 | 银行（杠杆10倍以上） |

**最好的ROE驱动是前两个**——靠真本事赚钱或高效运营，而不是靠借钱。

1. ✅ 三张表一起看，不要只看利润表
2. ✅ 重点关注经营现金流是否支撑利润
3. ✅ 对比同行业公司的关键指标
4. ✅ 留意财务造假的红旗信号
5. ✅ 用杜邦分析理解ROE的来源
