## 什么是RSI？

RSI（Relative Strength Index，相对强弱指标）由 J. Welles Wilder 发明，就像一个**市场情绪温度计**：

- 温度太高（RSI > 70）→ 市场过热，可能要降温回调
- 温度太低（RSI < 30）→ 市场过冷，可能要回暖反弹
- 温度适中（RSI 30-70）→ 市场运行正常

:::tip[形象理解]
RSI就像一个弹簧：
- 压得太紧（超卖）→ 弹簧要反弹
- 拉得太长（超买）→ 弹簧要回缩
- 正常范围内 → 弹簧可以继续压缩或拉伸
:::

## RSI的计算原理

RSI衡量的是**一段时间内上涨力度与下跌力度的比例**：

```
RSI = 100 - (100 / (1 + RS))
RS = N日内平均上涨幅度 / N日内平均下跌幅度
```

常用参数：**RSI(14)** — 14日RSI（默认参数）

- 上涨天数多、涨幅大 → RS大 → RSI趋近100
- 下跌天数多、跌幅大 → RS小 → RSI趋近0
- 涨跌相当 → RSI在50附近

## RSI的核心用法

### 用法一：超买超卖

**经典阈值（但不是绝对的！）：**

| RSI值 | 状态 | 含义 |
|---|---|---|
| **> 80** | 极度超买 | 短期涨过头了，回调风险大 |
| **70-80** | 超买区 | 上涨动能强，但要注意衰竭 |
| **50-70** | 偏多区 | 多头趋势中，正常上涨 |
| **30-50** | 偏空区 | 空头趋势中，正常下跌 |
| **20-30** | 超卖区 | 下跌动能强，但可能企稳 |
| **< 20** | 极度超卖 | 短期跌过头了，反弹概率大 |

:::caution[重要警告]
**超买不等于立刻跌，超卖不等于立刻涨！**

在强势上涨中，RSI可以长期停留在70以上。
在强势下跌中，RSI可以长期停留在30以下。

超买超卖只是**预警**，不是**入场信号**。必须等价格确认！
:::

### 用法二：RSI背离（最有价值！）

和MACD一样，RSI的**背离信号**非常珍贵。

**顶背离（看跌）：**
> 价格创新高，但RSI没有创新高

说明虽然价格还在涨，但上涨的"力度"在减弱。

**底背离（看涨）：**
> 价格创新低，但RSI没有创新低

说明虽然价格还在跌，但下跌的"力度"在减弱。

:::tip[背离的可靠性]
1. RSI背离在**极端区域**（超买/超卖区）更可靠
2. **二次背离**比一次背离更可靠
3. **周线RSI背离**比日线更可靠
4. 背离 + 超买/超卖区域 = 最强信号
:::

### 用法三：50线多空分界

- RSI在50**上方** → 多头占优
- RSI在50**下方** → 空头占优
- RSI反复穿越50 → 震荡市

### 用法四：RSI趋势线

在RSI指标上画趋势线（连接RSI的高点或低点），RSI趋势线的突破往往**提前于**价格趋势线的突破。

## RSI实战交易策略

### 策略一：超卖反弹

条件：
1. RSI跌破30进入超卖区
2. RSI拐头向上穿过30线
3. 价格出现止跌K线
4. 成交量放大配合

止损：RSI重新跌破30或价格创新低

### 策略二：RSI底背离买入

条件：
1. 价格创新低，RSI未创新低
2. RSI在30以下（超卖区域）
3. RSI金叉向上
4. 价格收出确认阳线

止损：背离最低点下方

### 策略三：RSI + 均线组合

最强确认组合：
1. 价格站上MA60（趋势确认）
2. RSI从超卖区回升到50以上（动量确认）
3. 成交量放大（资金确认）
→ 三重确认，可靠性极高

## RSI的常见参数选择

| 参数 | 特点 | 适用场景 |
|---|---|---|
| RSI(6) | 敏感，信号多但假信号也多 | 超短线 |
| **RSI(14)** | **平衡，最常用** | **波段交易** |
| RSI(21) | 平滑，信号少但可靠 | 中线交易 |

## 常见错误

### ❌ RSI到70就卖，到30就买

在强趋势中RSI可以长期停留在极端区域。机械地按70/30操作会被反复打脸。

✅ **结合趋势方向判断**：上升趋势中RSI到70可能只是"休息"，回调到40-50反而可能是买入机会。

### ❌ 忽略大趋势

周线RSI在70以上（强势），日线RSI到30可能只是小回调。

✅ **先看大周期RSI的方向，再在小周期找机会。**

## 实战检查清单

1. ✅ 先看周线RSI判断大趋势的多空状态
2. ✅ 日线RSI超卖 + 底背离 → 关注买入机会
3. ✅ 日线RSI超买 + 顶背离 → 考虑减仓
4. ✅ RSI信号必须配合价格确认和成交量
5. ✅ 不机械执行70/30规则，结合趋势判断
