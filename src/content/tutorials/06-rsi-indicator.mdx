## 什么是RSI？

RSI（Relative Strength Index，相对强弱指标），由技术分析大师 J. Welles Wilder Jr. 于1978年发明。

如果说MACD是趋势的"速度计"，那RSI就是市场的**温度计**——它能告诉你当前市场是"过热"还是"过冷"，帮助你判断价格是否偏离了合理范围。

:::tip[提示]
RSI = 测量"涨的力量"和"跌的力量"的比值。RSI越高，说明近期涨的力度越强；RSI越低，说明近期跌的力度越强。
:::

## RSI的计算原理

### 核心公式

```
RSI = 100 - 100 / (1 + RS)

其中：
RS = N日内平均上涨幅度 / N日内平均下跌幅度
```

### 详细计算步骤

假设我们用14天RSI（最常用的参数），来看一组实际数据：

**某股票最近14天的涨跌情况：**

| 日期 | 收盘价 | 涨跌 | 上涨幅度 | 下跌幅度 |
|---|---|---|---|---|
| 第1天 | 25.00 | - | - | - |
| 第2天 | 25.50 | +0.50 | 0.50 | 0 |
| 第3天 | 25.20 | -0.30 | 0 | 0.30 |
| 第4天 | 25.80 | +0.60 | 0.60 | 0 |
| 第5天 | 26.10 | +0.30 | 0.30 | 0 |
| 第6天 | 25.70 | -0.40 | 0 | 0.40 |
| 第7天 | 25.40 | -0.30 | 0 | 0.30 |
| 第8天 | 25.90 | +0.50 | 0.50 | 0 |
| 第9天 | 26.30 | +0.40 | 0.40 | 0 |
| 第10天 | 26.00 | -0.30 | 0 | 0.30 |
| 第11天 | 26.50 | +0.50 | 0.50 | 0 |
| 第12天 | 27.00 | +0.50 | 0.50 | 0 |
| 第13天 | 26.60 | -0.40 | 0 | 0.40 |
| 第14天 | 27.20 | +0.60 | 0.60 | 0 |

**步骤一：计算平均上涨和平均下跌**

```
14天内上涨天数 = 9天
上涨幅度总和 = 0.50 + 0.60 + 0.30 + 0.50 + 0.40 + 0.50 + 0.50 + 0.60 = 3.90
平均上涨 = 3.90 / 14 = 0.279

14天内下跌天数 = 5天
下跌幅度总和 = 0.30 + 0.40 + 0.30 + 0.30 + 0.40 = 1.70
平均下跌 = 1.70 / 14 = 0.121
```

**步骤二：计算RS和RSI**

```
RS = 0.279 / 0.121 = 2.306
RSI = 100 - 100 / (1 + 2.306) = 100 - 30.3 = 69.7
```

**解读：** RSI = 69.7，接近70的超买线，说明近期上涨力度较强，但尚未进入极端超买区域。

## RSI的核心用法

### 用法一：超买超卖判断

这是RSI最基础也是最广为人知的用法：

| RSI范围 | 状态 | 含义 | 操作建议 |
|---|---|---|---|
| **> 70** | 超买 | 涨太多了，可能要回调 | 注意风险，考虑减仓 |
| **50-70** | 偏强 | 多头占优，趋势向上 | 可以持股或逢低买入 |
| **30-50** | 偏弱 | 空头占优，趋势向下 | 谨慎观望 |
| **< 30** | 超卖 | 跌太多了，可能要反弹 | 关注买入机会 |

:::caution[注意]
**这是RSI最大的误区！**

- 超买（RSI > 70）只说明"短期涨得比较多"，**不代表立刻下跌**
- 在强势上涨中，RSI可能长时间停留在70以上（甚至80以上）
- 超买之后还可以更超买——不要仅因为RSI > 70就做空

正确做法：超买/超卖只是预警，需要配合其他信号确认。
:::

### 用法二：RSI背离（最值钱的信号）⭐

和MACD一样，RSI的背离信号非常珍贵。

**顶背离（看跌信号）：**
> 股价创新高，但RSI没有创新高

说明虽然价格还在涨，但上涨的"力度"在减弱——多头累了。

**底背离（看涨信号）：**
> 股价创新低，但RSI没有创新低

说明虽然价格还在跌，但下跌的"力度"在减弱——空头快没力气了。

**背离的可靠性排序：**
1. **周线背离** > 日线背离 > 60分钟背离（周期越大越可靠）
2. **二次背离** > 一次背离（多次背离更可靠）
3. **RSI极端区域的背离** > 中间区域的背离

### 用法三：RSI趋势线

可以在RSI指标上画趋势线，效果往往比在价格上画更清晰：

- RSI突破下降趋势线 → 买入信号
- RSI跌破上升趋势线 → 卖出信号

### 用法四：50线多空分界

RSI的50线是重要的多空分界线：

- RSI从下往上穿越50 → 多头转强，可以考虑买入
- RSI从上往下跌破50 → 空头转强，考虑卖出或减仓

## RSI实战交易策略

### 策略一：超卖反弹

**条件：**
1. RSI跌到30以下（超卖）
2. RSI开始拐头向上，突破30
3. 价格出现止跌K线（锤子线、早晨之星等）
4. 成交量放大

**入场：** RSI突破30时买入
**止损：** RSI回到30以下或价格创新低
**止盈：** RSI达到60-70区域

**模拟案例：** 某医药股连续下跌，RSI降至22。随后出现锤子线，RSI拐头向上突破30。此时可以考虑轻仓买入，止损设在近期低点下方3%。

### 策略二：底背离买入

**条件：**
1. 价格创新低，但RSI没有创新低
2. RSI从30以下开始回升
3. 价格出现止跌信号
4. MACD同时出现底背离（共振信号）

**入场：** 价格突破底背离时的反弹高点
**止损：** 背离时的最低点下方
**止盈：** RSI达到70以上或出现顶背离

**模拟案例：** 某新能源股从40元跌到32元（第一波低点，RSI=25），反弹到35元后再次跌到31元（第二波低点，RSI=28）。价格创新低但RSI没有创新低 = 底背离。随后股价出现大阳线突破33元，这是买入信号。

### 策略三：50线突破

**条件：**
1. RSI长期在50以下运行
2. RSI突破50并站稳
3. 成交量放大配合
4. 均线开始多头排列

**入场：** RSI突破50后回踩不破50时买入
**止损：** RSI跌破50
**止盈：** RSI达到70以上

## 不同参数的RSI

| 参数 | 适用场景 | 特点 |
|---|---|---|
| **RSI(6)** | 短线交易 | 信号灵敏，但假信号多 |
| **RSI(14)** | 通用（推荐） | 信号均衡，最常用 |
| **RSI(21)** | 中线交易 | 信号稳定，但滞后 |
| **RSI(25)** | 长线投资 | 非常稳定，信号少 |

:::tip[提示]
对于大多数投资者，**RSI(14)** 是最佳选择。这是Wilder最初推荐的参数，经过几十年验证。

短线交易者可以同时看RSI(6)和RSI(14)，当两者同时超买或超卖时信号更可靠。
:::

## RSI + MACD 组合使用

RSI和MACD是互补的指标，组合使用效果最好：

| 场景 | MACD信号 | RSI信号 | 可靠性 |
|---|---|---|---|
| **强势买入** | 零轴上方金叉 | RSI从50以下突破50 | ⭐⭐⭐⭐⭐ |
| **底背离买入** | 底背离 | 底背离 | ⭐⭐⭐⭐⭐ |
| **超卖反弹** | DIF拐头向上 | RSI从30以下回升 | ⭐⭐⭐⭐ |
| **趋势确认** | 多头排列 | RSI在50-70之间 | ⭐⭐⭐⭐ |
| **顶背离卖出** | 顶背离 | 顶背离 | ⭐⭐⭐⭐⭐ |

## RSI的常见错误

### ❌ 错误一：RSI超买就卖，超卖就买

在强趋势中，RSI可以长期停留在超买/超卖区域。

✅ **正确做法：** 超买/超卖只做预警，等RSI从超买/超卖区域走出来后再操作。

### ❌ 错误二：所有周期用同样的超买超卖标准

日线RSI > 70是超买，但周线RSI > 70的意义完全不同。

✅ **正确做法：** 周线RSI的超买标准应该更严格（如80以上），因为周线级别超买代表中长期过热。

### ❌ 错误三：忽略市场环境

在牛市中，RSI经常在50-80之间运行；在熊市中，RSI经常在20-50之间运行。

✅ **正确做法：** 先判断大趋势，再在趋势方向上使用RSI信号。牛市中关注超卖反弹信号，熊市中关注超买回落信号。

## RSI在不同市场环境下的表现

| 市场环境 | RSI表现 | 使用建议 |
|---|---|---|
| **强趋势上涨** | 长期在50-80之间 | 关注RSI回到50附近的买入机会 |
| **强趋势下跌** | 长期在20-50之间 | 关注RSI反弹到50附近的减仓机会 |
| **宽幅震荡** | 超买超卖交替出现 | RSI最好用的环境 |
| **窄幅震荡** | 在40-60之间波动 | 信号不明显，减少使用 |

## 实战检查清单

1. ✅ 先看大趋势方向（均线、MACD）
2. ✅ RSI超买/超卖只是预警，不等同于买卖信号
3. ✅ 重点关注RSI背离——这是最有价值的信号
4. ✅ RSI突破50是多空转换的重要参考
5. ✅ 配合MACD一起使用，信号共振更可靠
6. ✅ 周线RSI信号 > 日线RSI信号
7. ✅ 震荡市用RSI效果好，趋势市要谨慎
