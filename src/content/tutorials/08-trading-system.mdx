## 为什么需要交易系统？

想象两个场景：

**没有交易系统的人：**
> "今天看到这只股票涨了，买一点试试……"
> "哎呀跌了，要不要卖呢？再等等吧……"
> "又跌了！完了完了，亏太多了……"
> "终于涨回来了，赶紧卖！……怎么还在涨？"

**有交易系统的人：**
> "按照我的选股标准，这只股票符合条件。"
> "入场价25元，止损23.5元，目标28元。风险收益比约1:2，值得做。"
> "跌到止损位了，执行止损。亏损在可控范围内。"
> "复盘记录：这笔交易逻辑没错，执行也到位。下次继续。"

**区别：** 前者在赌博，后者在做投资。

:::tip[提示]
交易系统 = 你的投资宪法。它规定了什么时候买、什么时候卖、买多少、错了怎么办。

没有系统的人，每次交易都在重新做决策 → 疲惫、焦虑、容易犯错。
有系统的人，只需要执行规则 → 轻松、理性、长期稳定盈利。
:::

## 交易系统的五大模块

一个完整的交易系统包含五个环节，每个环节都有明确的规则：

```
选股 → 分析 → 决策 → 执行 → 复盘
 ↑                                |
 └────── 优化系统 ←───────────────┘
```

### 模块一：选股规则

**定义：** 什么样的股票才值得关注？

选股规则应该明确、可量化、可执行。

**示例——趋势跟踪选股规则：**
1. 日线MA20以上（中期趋势向上）
2. 日线成交量 > 20日均量的1.2倍（有资金关注）
3. 行业处于景气周期（基本面配合）
4. 市值 > 100亿（避免小盘股流动性风险）
5. 非ST股，非次新股（上市满1年）

**示例——价值投资选股规则：**
1. ROE连续3年 > 15%（赚钱效率高）
2. PE < 同行业中位数（相对便宜）
3. PEG < 1.5（增长合理）
4. 负债率 < 60%（财务健康）
5. 连续3年分红（重视股东回报）

:::note[备注]
选股规则不在于多复杂，而在于**你能坚持执行**。简单的规则 + 长期执行 > 复杂的规则 + 三天打鱼两天晒网。
:::

### 模块二：分析框架

**定义：** 选出股票后，怎么分析它值不值得买？

**四维分析框架：**

**1. 技术面（什么时候买？）**
- 趋势方向（均线排列、MACD方向）
- 关键支撑/压力位
- 当前处于什么形态

**2. 基本面（值不值得买？）**
- 公司的核心竞争力
- 行业景气度
- 财务数据是否健康

**3. 资金面（有没有人买？）**
- 近期成交量变化
- 主力资金动向
- 北向资金/融资余额变化

**4. 市场面（环境对不对？）**
- 大盘趋势如何
- 板块轮动方向
- 市场情绪偏乐观还是悲观

### 模块三：决策规则

**定义：** 满足什么条件才买入/卖出？

**买入决策模板：**

| 条件 | 要求 | 当前状态 |
|---|---|---|
| 趋势方向 | MA20以上 | ✅/❌ |
| 买入信号 | MACD金叉或突破前高 | ✅/❌ |
| 成交量 | 放量确认 | ✅/❌ |
| 基本面 | 无重大利空 | ✅/❌ |
| 大盘环境 | 非系统性风险 | ✅/❌ |

**至少满足4个条件才允许买入。**

**卖出决策模板：**

| 触发条件 | 操作 | 优先级 |
|---|---|---|
| 跌破止损位 | 无条件卖出 | 最高 |
| 买入逻辑被推翻 | 尽快卖出 | 高 |
| 达到目标价1 | 卖出50% | 中 |
| 达到目标价2 | 全部卖出 | 中 |
| 出现顶背离 | 减仓50% | 中 |
| 时间止损（N天不涨） | 卖出 | 低 |

### 模块四：执行纪律

**定义：** 怎么确保你按规则操作？

**执行三原则：**

**1. 交易前写计划**
每笔交易买入前必须写下：
- 买入理由（不超过3条）
- 买入价格区间
- 止损价格
- 目标价格
- 仓位比例

**2. 用条件单代替手动**
- 提前设好止损单
- 提前设好止盈单
- 减少盘中情绪干扰

**3. 盘中不看账户盈亏**
- 只看K线和信号
- 盈亏数字会影响判断
- 信号到了就执行，不管盈亏

### 模块五：复盘机制

**定义：** 每次交易后怎么总结和改进？

每次交易结束后回答这些问题：
1. 这笔交易赚钱还是亏钱？
2. 买入逻辑对不对？（事后再看）
3. 执行是否到位？（有没有提前卖或扛着不卖？）
4. 有什么可以改进的地方？
5. 系统规则需要调整吗？

## 三种经典交易系统模板

### 系统一：趋势跟踪系统

**核心理念：** 顺势而为，截断亏损，让利润奔跑

**规则：**
- **选股：** MA20以上 + 成交量放大
- **买入：** 价格回调到MA20附近 + MACD金叉
- **止损：** 跌破MA20（收盘价确认）
- **止盈：** 移动止损（最高价回撤8%）
- **仓位：** 单只不超过20%，总仓位根据大盘状态调整

**适合谁：** 有耐心、能接受较高回撤、中线投资者

**优势：** 趋势行情中收益可观
**劣势：** 震荡市中反复止损

### 系统二：区间交易系统

**核心理念：** 在支撑位买、压力位卖，赚波动的钱

**规则：**
- **选股：** 在明确区间内震荡的股票
- **买入：** 价格触及区间下沿 + RSI < 30
- **止损：** 跌破区间下沿3%
- **止盈：** 价格到达区间上沿
- **仓位：** 每次固定仓位（如总资金的10%）

**适合谁：** 喜欢短线操作、盯盘时间多的人

**优势：** 震荡市表现好、胜率高
**劣势：** 一旦突破区间可能连续亏损

### 系统三：价值投资系统

**核心理念：** 以合理价格买入优秀公司，长期持有

**规则：**
- **选股：** ROE > 15% + 行业龙头 + 竞争壁垒强
- **买入：** 估值低于历史中位数（PE/PB处于30%分位以下）
- **止损：** 基本面恶化（ROE连续下降、核心逻辑被推翻）
- **止盈：** 严重高估（PE处于历史90%分位以上）
- **仓位：** 集中持有5-10只，长期满仓

**适合谁：** 不想频繁交易、能长期持有的人

**优势：** 省心、长期收益稳定
**劣势：** 需要深入研究基本面，持股周期长

## 从零搭建系统的步骤

### 步骤一：确定你的交易风格

| 问题 | 选项A | 选项B | 选项C |
|---|---|---|---|
| 你能盯盘吗？ | 能（日内） | 偶尔（波段） | 不能（长线） |
| 你能承受多大亏损？ | 5%（短线） | 10%（波段） | 20%（长线） |
| 你偏好哪种分析？ | 技术面 | 技术面+基本面 | 基本面 |
| 你的资金量？ | 小资金 | 中等 | 大资金 |

### 步骤二：写下你的规则

用一张纸写下：
1. 什么条件的股票我才会买？
2. 什么价格买入？
3. 买多少？
4. 什么条件下止损？
5. 什么条件下止盈？

### 步骤三：模拟测试

用模拟盘或小资金测试你的系统：
- 至少测试30笔交易
- 记录每笔交易的详细信息
- 计算胜率、平均盈利、平均亏损、盈亏比

### 步骤四：评估和优化

**核心指标：**

| 指标 | 合格标准 | 优秀标准 |
|---|---|---|
| **胜率** | > 40% | > 55% |
| **盈亏比** | > 1.5:1 | > 2:1 |
| **最大回撤** | < 20% | < 10% |
| **月均收益** | > 0 | > 2% |

**如果系统不合格：**
- 胜率低 → 优化选股条件或买入时机
- 盈亏比低 → 让盈利更充分（扩大止盈）或减少亏损（缩小止损）
- 回撤大 → 减少单笔仓位或增加止损严格度

### 步骤五：严格执行

系统建立后，最重要的一步：**执行**。

每次想违反规则时问自己：
- "我是在执行系统，还是在凭感觉？"
- "如果这样做亏了，我能接受吗？"

## 常见的系统缺陷

### 缺陷一：规则太模糊

❌ "我觉得能涨就买"
✅ "MA20以上 + MACD金叉 + 成交量放大50%以上"

### 缺陷二：没有止损规则

❌ "跌了就等着"
✅ "收盘价跌破MA20 → 第二天开盘卖出"

### 缺陷三：过度优化

❌ 为了让历史数据好看，把参数调到完美
✅ 用简单的参数，在不同市场环境下都能稳定

### 缺陷四：频繁修改规则

❌ 亏了一笔就改规则
✅ 至少积累30笔交易数据后再评估是否需要改

## 交易系统检查清单

1. ✅ 有明确的选股规则（可量化、可执行）
2. ✅ 有明确的买入条件（至少3个条件）
3. ✅ 有明确的止损规则（每笔交易必须设）
4. ✅ 有明确的止盈规则（分批止盈更优）
5. ✅ 有仓位管理规则（单笔、总仓位上限）
6. ✅ 有复盘机制（每笔交易都记录）
7. ✅ 规则简单到你能坚持执行
