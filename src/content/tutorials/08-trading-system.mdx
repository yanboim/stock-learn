## 为什么需要交易系统？

想象两个场景：

**没有交易系统的小明：**
> 今天听朋友说某股要涨 → 冲进去买了 → 涨了很高兴加仓 → 跌了恐慌卖出 → 亏损20% → 又听另一个消息...

**有交易系统的小红：**
> 按照选股条件筛选 → 等待买入信号出现 → 按计划仓位入场 → 设好止损止盈 → 到了就执行，不犹豫

**区别在于：小明靠情绪和运气，小红靠规则和概率。**

:::tip[核心理念]
交易系统 = 一套**明确的规则**，告诉你：
1. **买什么**（选股）
2. **什么时候买**（入场时机）
3. **买多少**（仓位管理）
4. **什么时候卖**（出场规则）
5. **错了怎么办**（风险控制）
:::

## 实战：搭建一个简单有效的交易系统

### 示例系统："趋势跟随+估值保护"

**选股规则：**
- ROE连续3年 > 15%
- PE(TTM) < 行业PE中位数
- 营收同比增长 > 10%
- 非ST股，日均成交额 > 5000万

**入场规则（全部满足才买）：**
1. 周线MACD在零轴上方（大趋势向上）
2. 日线MACD金叉（中期信号）
3. 价格站上MA20（短期确认）
4. 成交量放大至5日均量1.5倍以上

**仓位管理：**
- 单只股票最多占总资金20%
- 同时持有最多5只股票
- 分2批建仓：信号出现买50%，突破确认加50%
- 总仓位根据市场环境调整：
  - 牛市/强势：80-100%
  - 震荡市：50-70%
  - 熊市/弱势：20-40%或空仓

**出场规则：**
- 止盈：涨幅达20-30%减仓一半，剩下的跟踪止损
- 止损：跌破MA20减仓50%，跌破MA60清仓
- 逻辑止损：买入逻辑被证伪（如业绩暴雷）

**风控规则：**
- 单笔最大亏损：总资金的3%
- 月度最大回撤：8%
- 连续3笔亏损 → 暂停1周复盘

## 不同风格的交易系统

### 波段交易系统（2周-3个月）

| 要素 | 波段系统 |
|---|---|
| 选股 | 趋势向上 + 基本面不差 |
| 入场 | 日线MACD金叉 + 放量 |
| 持仓 | 2-6周 |
| 出场 | 目标价到达或止损触发 |
| 适合人群 | 有一定经验，能每天看盘 |

### 中线投资系统（3-12个月）

| 要素 | 中线系统 |
|---|---|
| 选股 | 优质公司 + 估值合理 |
| 入场 | 季度财报确认 + 技术底部 |
| 持仓 | 3-12个月 |
| 出场 | 估值过高或基本面恶化 |
| 适合人群 | 上班族，没时间天天看盘 |

### 长线价值投资（1年以上）

| 要素 | 长线系统 |
|---|---|
| 选股 | 行业龙头 + 护城河深 |
| 入场 | 估值处于历史低位区间 |
| 持仓 | 1年以上 |
| 出场 | 公司核心竞争力丧失 |
| 适合人群 | 耐心强，关注基本面 |

## 交易系统最重要的品质：一致性

一套好的交易系统**不需要每次都赢**，但必须：

1. **可重复** — 每次执行的条件相同
2. **可衡量** — 能统计胜率和盈亏比
3. **可改进** — 有明确的复盘方法

:::tip[关键数字]
一套胜率只有40%的系统，如果平均盈利是平均亏损的2倍，长期仍然赚钱：
- 10次交易：4次赚 × 2 = 8，6次亏 × 1 = 6
- 净利润 = 8 - 6 = **+2个单位**

**盈亏比比胜率更重要！**
:::

## 常见错误

### ❌ 频繁更换系统

今天用均线系统，明天用TD系统，后天又换成价值投资。

✅ 选定一套系统至少执行3个月，用数据说话。

### ❌ 系统太复杂

要求10个条件全部满足才交易 → 可能一年都找不到机会。

✅ 简单的系统更容易坚持执行。3-5个核心条件就够。

### ❌ 只优化不执行

一直在完善系统参数，却从不下场交易。

✅ 先用最简单的版本开始，在实战中逐步改进。
