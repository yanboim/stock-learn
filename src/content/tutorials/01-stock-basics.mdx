## 什么是股票？

想象一下：你的朋友开了一家奶茶店，生意非常好，他想开分店但钱不够。他说："你出10万块，我给你店里10%的股份，以后赚了钱按比例分给你。"

**这10%的股份，本质上就是"股票"。**

股票 = 公司所有权的一小部分。买股票 = 你成了这家公司的股东。

### 股票的核心特点

- **所有权凭证**：持有股票意味着你拥有公司的一部分
- **收益来源**：股价上涨赚取差价 + 公司分红
- **风险共担**：公司赚钱你赚钱，公司亏钱你也亏
- **流动性**：可以在交易时间内随时买卖（不像房产很难变现）

:::tip[一句话理解]
买股票就是买公司。你买的不是一串代码，而是一家真实企业的一部分。
:::

### 股票 vs 其他投资方式

| 投资方式 | 预期年化收益 | 风险等级 | 流动性 | 适合谁 |
|---|---|---|---|---|
| **银行存款** | 1.5-2.5% | 极低 | 高 | 不想承担任何风险的人 |
| **货币基金** | 1.5-2.5% | 极低 | 高 | 闲钱理财 |
| **债券基金** | 3-5% | 低 | 中 | 稳健型投资者 |
| **股票** | 8-15%（长期） | 中高 | 高 | 愿意学习、能承受波动 |
| **房产** | 5-10% | 中 | 低 | 资金量大、长期持有 |
| **加密货币** | 波动极大 | 极高 | 高 | 高风险偏好 |

**关键区别：** 股票是普通人能接触到的、收益潜力最高的投资工具之一，但前提是你必须学会控制风险。

## 中国A股市场结构

中国的股票市场就像一个大型交易市场，分为几个"摊位区域"：

### 交易所

| 交易所 | 成立时间 | 主要特点 |
|---|---|---|
| **上海证券交易所（上交所）** | 1990年 | 大型国企、蓝筹股为主，主板+科创板 |
| **深圳证券交易所（深交所）** | 1991年 | 中小企业为主，主板+创业板 |
| **北京证券交易所（北交所）** | 2021年 | 专精特新中小企业 |

### 股票板块

- **主板** — 大型成熟企业，门槛最低（60xxxx 上交所，00xxxx 深交所）
- **科创板** — 科技创新企业，上交所（688xxx），需50万+2年经验
- **创业板** — 成长型创新企业，深交所（300xxx），需10万+2年经验
- **北交所** — 中小企业（8xxxxx），需50万+2年经验

### 股票代码速查

```

002xxx                    → 中小板（已并入主板）

```

## 交易规则：A股怎么玩？

### 基本规则

- **交易时间**：周一至周五（法定节假日除外）
  - 上午 9:30 - 11:30
  - 下午 13:00 - 15:00
- **交易单位**：最少买 **100股**（1手），卖出不限
- **涨跌幅限制**：主板 ±10%，科创板/创业板 ±20%
- **T+1 制度**：今天买入的股票，**明天才能卖出**
- **竞价方式**：
  - 9:15-9:25 集合竞价（确定开盘价）
  - 9:30-11:30、13:00-14:57 连续竞价
  - 14:57-15:00 收盘集合竞价

:::caution[T+1 的重要性]
T+1意味着你今天买入后如果股价大跌，**当天无法卖出止损**。这就是为什么控制仓位和风险特别重要！
:::

### 交易费用详解

每次买卖股票都要付手续费，别小看这些费用：

| 费用项目 | 费率 | 最低收费 | 谁收的 |
|---|---|---|---|
| **佣金** | 万2.5左右（可谈） | 5元 | 券商 |
| **印花税** | 卖出时收取 0.05% | 无 | 国家 |
| **过户费** | 0.001% | 无 | 交易所 |

#### 费用计算实例

**买入10,000元的股票：**
```
佣金：10,000 × 0.025% = 2.5元 → 不足5元按5元收取 = 5元
过户费：10,000 × 0.001% = 0.1元
买入总费用 = 5.1元
```

**卖出10,000元的股票（假设涨到11,000元）：**
```
佣金：11,000 × 0.025% = 2.75元 → 不足5元按5元收取 = 5元
印花税：11,000 × 0.05% = 5.5元
过户费：11,000 × 0.001% = 0.11元
卖出总费用 = 10.61元
```

**一笔完整的买卖（10,000元买入，11,000元卖出）：**
```
利润：1,000元
总费用：5.1 + 10.61 = 15.71元
实际到手：984.29元
费用占比：1.57%
```

:::tip[省钱技巧]
佣金是可以和券商谈的！资金量大可以谈到万1甚至更低。印花税是国家收的，没法省。

**重要：** 如果每次交易金额太小（比如只买1,000元），5元最低佣金会吃掉大量利润。建议每次交易至少2万元以上才划算。
:::

### 集合竞价怎么定价？

集合竞价是A股特有的定价机制，理解它对你抓住开盘机会很重要：

**9:15-9:20**：可以挂单也可以撤单（有些人故意挂大单然后撤掉，制造假象）
**9:20-9:25**：只能挂单不能撤单（这时候看到的委托是真实的）
**9:25**：系统计算出一个价格，让最多成交量成交 → 这个就是开盘价

**开盘价的意义：**
- 高开（开盘价 > 昨日收盘价）→ 市场偏乐观
- 低开（开盘价 < 昨日收盘价）→ 市场偏悲观
- 平开 → 多空均衡

## 怎么开户？

现在开户非常简单，手机上就能完成：

### 选择券商的考虑因素

| 因素 | 重要性 | 建议 |
|---|---|---|
| **佣金费率** | ⭐⭐⭐ | 选万1.5以下的，大资金可谈更低 |
| **APP体验** | ⭐⭐⭐⭐ | 好用的APP能提升交易效率 |
| **研报服务** | ⭐⭐ | 大券商研报质量更高 |
| **客服响应** | ⭐⭐ | 新手更需要好的客服支持 |
| **营业部距离** | ⭐ | 大部分业务线上就能办 |

**推荐：** 华泰证券、中信证券、国泰君安、东方财富都是不错的选择。

### 开户步骤

1. **选券商** — 建议选大型券商
2. **下载APP** — 各券商都有自己的手机APP
3. **在线开户** — 身份证 + 银行卡 + 人脸识别，10分钟搞定
4. **等待审核** — 一般1个工作日内完成
5. **银证转账** — 把银行卡的钱转到证券账户
6. **开始交易** — 建议先用少量资金练手

:::note[建议]
新手建议同时开 **A股账户** + **基金账户**。有些券商还送Level-2行情（可以看到买卖盘口细节），开户前可以多比较。
:::

## A股市场的关键历史事件

了解A股的历史，能帮你理解市场的脾气：

| 时间 | 事件 | 教训 |
|---|---|---|
| **2007年** | 上证指数冲到6124点历史高位 | 牛市顶峰，之后暴跌至1664点 |
| **2015年** | 杠杆牛到5178点后崩盘 | 杠杆放大收益也放大亏损，千股跌停 |
| **2019年** | 科创板开板 | 注册制改革，新股发行更市场化 |
| **2020年** | 疫情后A股快速反弹 | 市场对突发事件反应剧烈但会修复 |
| **2024年** | 新"国九条"发布 | 加强监管、鼓励分红、退市制度完善 |

:::caution[历史教训]
A股有一个特点：**牛短熊长**。牛市来得快去得也快，熊市则漫长难熬。
- 牛市时不要过度乐观（不要加杠杆）
- 熊市时不要过度悲观（好公司会更便宜）
:::

## 新手最容易犯的5个错误

### ❌ 错误一：追涨杀跌

看到股票涨了就追进去，跌了就恐慌卖出。结果：**高买低卖，反复亏损**。

**数据说话：** 散户追涨杀跌的典型模式是——股票涨了20%才追进去，跌了5%就割肉。反复操作几次，本金就损失大半。

✅ **正确做法**：制定买入和卖出计划，不因短期波动改变策略。

### ❌ 错误二：满仓一只股票

把所有钱押在一只股票上，相当于赌博。

✅ **正确做法**：分散投资，至少持有3-5只不同行业的股票。单只股票不超过总资金的15-20%。

### ❌ 错误三：听消息炒股

"我朋友说这只股票要涨" → 结果往往是最后接盘的人。

**真相：** 当消息传到你耳朵里时，通常已经涨完了。等你买入正好是别人卖出的时候。

✅ **正确做法**：自己做研究，看公司基本面、财报、行业趋势。

### ❌ 错误四：频繁交易

每天都想操作，手续费就吃掉了利润。

**算一笔账：**
```
每天交易一次，每次交易1万元：
买入费用：5元
卖出费用：约10元
每天费用：15元
一个月（22个交易日）：330元
一年：3,960元

如果你的本金是10万，一年手续费就吃掉4%！
这还不算你亏损的钱。
```

✅ **正确做法**：减少交易频率，每次交易都要有明确的理由。

### ❌ 错误五：不设止损

亏损了不卖，觉得"总会涨回来的"。结果越套越深。

**残酷事实：** 从48元跌到8元的中石油，已经17年了还没回本。

✅ **正确做法**：每笔交易前设好止损位，到了就坚决执行。

## 建立正确的投资心态

### 心态一：接受亏损是正常的

没有人能100%正确。专业投资者的胜率通常也只有40-60%，但他们靠的是：
- 赚的时候赚得多
- 亏的时候亏得少
- 长期坚持正期望的系统

### 心态二：投资是马拉松，不是百米冲刺

年化10-15%已经是非常优秀的投资业绩。
- 10万本金，年化12%，30年后 = **约300万**
- 但如果你中途因为追涨杀跌亏了50%，需要翻倍才能回本

### 心态三：独立思考，不要从众

当所有人都在讨论股票时，往往是最危险的时候。
当所有人都不想谈股票时，往往是机会来临的时候。

## 投资前的自我检查清单

开始投资前，先问自己这些问题：

1. ✅ 我有3-6个月的应急备用金吗？
2. ✅ 我投入的钱是闲钱吗（即使全亏也不影响生活）？
3. ✅ 我了解基本的风险吗？
4. ✅ 我有明确的投资策略吗？
5. ✅ 我知道什么时候止损吗？
6. ✅ 我不会因为一笔亏损就影响正常生活吗？
7. ✅ 我能承受50%的账面浮亏吗？

:::caution[最重要的话]
**永远不要借钱炒股，永远不要用生活必需的钱炒股。**

投资的第一原则是保住本金，第二原则是记住第一条。
:::
