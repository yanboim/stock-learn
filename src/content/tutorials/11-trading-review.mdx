## 为什么复盘比交易更重要？

交易是执行，复盘是进化。

一笔交易赚了钱，如果你不知道为什么赚的，那和赌博没有区别。一笔交易亏了钱，如果你不知道为什么亏的，下次还会犯同样的错。

**复利的真正来源不是资金的增长，而是认知的提升。**

:::tip[提示]
每个职业运动员都会看比赛录像，分析每一次得失。交易复盘就是你的"比赛录像"——帮你看到交易时看不到的东西。
:::

## 交易日志：复盘的基础

### 每笔交易必须记录的信息

#### 基础信息

| 字段 | 示例 | 说明 |
|---|---|---|
| 股票名称 | 某新能源公司 | 交易标的 |
| 股票代码 | 300xxx | 方便回查 |
| 交易方向 | 买入/卖出 | 做多还是减仓 |

#### 买入记录

| 字段 | 示例 | 说明 |
|---|---|---|
| **买入日期** | 2025-03-15 | 精确到日 |
| **买入价格** | 25.30元 | 实际成交价 |
| **买入仓位** | 15%（1.5万） | 占总资金比例 |
| **买入理由** | MA20以上+MACD金叉+放量 | 不超过3条 |
| **止损价格** | 23.50元 | 入场前必须设好 |
| **目标价1** | 27.50元 | 第一目标 |
| **目标价2** | 30.00元 | 第二目标 |
| **风险收益比** | 1:1.2 ~ 1:2.5 | 止损距离 vs 目标距离 |
| **当时大盘** | 震荡偏强 | 大环境判断 |
| **当时情绪** | 平静/兴奋/焦虑 | 记录心理状态 |

#### 卖出记录

| 字段 | 示例 | 说明 |
|---|---|---|
| **卖出日期** | 2025-04-10 | 精确到日 |
| **卖出价格** | 28.20元 | 实际成交价 |
| **卖出理由** | 达到目标价1+时间考虑 | 为什么卖 |
| **持有天数** | 26天 | 交易周期 |
| **盈亏金额** | +1,735元 | 绝对金额 |
| **盈亏比例** | +11.6% | 百分比收益 |

#### 复盘评价

| 字段 | 示例 | 说明 |
|---|---|---|
| **买入决策评分** | 8/10 | 选股和时机好不好 |
| **执行评分** | 7/10 | 有没有按计划操作 |
| **卖出评分** | 6/10 | 卖得早了还是晚了 |
| **总结经验** | 入场时机好但卖早了 | 一句话总结 |
| **下次改进** | 持股耐心要更好 | 具体可执行的改进 |

## 📋 复盘模板

### 模板一：单笔交易复盘

每笔交易结束后，回答以下10个问题：

**1. 买入回顾**
- 我为什么买这只股票？（原始理由）
- 买入理由是否成立？（事后验证）
- 有没有遗漏重要的风险因素？

**2. 持有过程**
- 持有期间有没有触发止损？（有没有及时执行？）
- 持有期间情绪波动大吗？（什么时候最焦虑？）
- 有没有做过计划外的操作？（临时加仓、减仓等）

**3. 卖出回顾**
- 卖出理由是什么？（计划内还是情绪驱动？）
- 卖早了还是卖晚了？（和计划对比）
- 如果重来一次，我会做什么不同？

**4. 量化评价**
- 这笔交易的风险收益比实际是多少？
- 最大浮盈和最大浮亏各是多少？
- 占总资金的盈亏贡献是多少？

### 模板二：周度复盘

每周日花30分钟做：

**1. 本周交易统计**
```
交易次数：X笔
盈利笔数：X笔（胜率XX%）
平均盈利：+X%
平均亏损：-X%
净盈亏：+/-X%
最大单笔盈利：+X%
最大单笔亏损：-X%
```

**2. 本周最佳交易**
- 哪笔交易做得最好？
- 好在哪里？（选股好？执行好？卖出好？）
- 能复制吗？

**3. 本周最差交易**
- 哪笔交易做得最差？
- 差在哪里？（选股差？执行差？情绪失控？）
- 怎么避免？

**4. 本周系统评估**
- 系统规则有没有被违反？
- 有没有出现系统外的交易（冲动交易）？
- 需要调整什么规则？

### 模板三：月度复盘

每月末花1小时做：

**1. 月度业绩数据**

| 指标 | 本月 | 上月 | 目标 |
|---|---|---|---|
| 总收益率 | +X% | +X% | > 2% |
| 胜率 | XX% | XX% | > 50% |
| 盈亏比 | X:1 | X:1 | > 2:1 |
| 最大回撤 | -X% | -X% | < 5% |
| 交易次数 | X次 | X次 | < 10次 |
| 最佳交易 | +X% | +X% | - |
| 最差交易 | -X% | -X% | - |

**2. 收益曲线分析**
- 本月收益是平稳增长还是大起大落？
- 亏损主要来自哪几笔交易？
- 盈利主要来自哪几笔交易？

**3. 交易行为分析**
- 有没有追涨杀跌？
- 有没有扛单不止损？
- 有没有频繁交易？
- 有没有计划外交易？

**4. 系统优化方向**
- 哪些规则执行得好？
- 哪些规则需要修改？
- 下个月重点改进什么？

## 从复盘中发现规律

### 规律一：你的盈利模式是什么？

统计所有盈利交易的共同点：
- 是不是某个行业/板块的交易特别赚钱？
- 是不是某种技术形态的交易胜率特别高？
- 是不是某个时间段的表现特别好？

→ **找到你的优势，然后集中火力。**

### 规律二：你的亏损模式是什么？

统计所有亏损交易的共同点：
- 是不是追高买入的交易亏得最多？
- 是不是某类股票的交易总亏钱？
- 是不是情绪化交易的亏损最大？

→ **找到你的弱点，然后主动规避。**

### 规律三：你的交易频率合适吗？

| 交易频率 | 可能问题 |
|---|---|
| **每月 > 20次** | 过于频繁，手续费和精力消耗大 |
| **每月 10-20次** | 偏多，可以减少低质量交易 |
| **每月 3-10次** | 适中，注重质量 |
| **每月 < 3次** | 可能过于谨慎，错过机会 |

## 复盘的四个层次

### 第一层：看盈亏（入门）
- 赚了还是亏了？
- 赚多少亏多少？

### 第二层：看执行（基础）
- 有没有按计划操作？
- 止损执行了没有？
- 有没有情绪化操作？

### 第三层：看逻辑（进阶）
- 买入逻辑对不对？
- 如果重来一次，同样的逻辑还值得做吗？
- 系统规则是否需要调整？

### 第四层：看自己（高级）
- 我在什么情况下容易犯错？
- 我的心理弱点是什么？
- 我能做什么来改善交易心理？

## 常见复盘错误

### ❌ 错误一：只复盘亏损，不复盘盈利

盈利交易也可能有问题：
- 可能是运气赚的钱（不可复制）
- 可能卖得太早（本可以赚更多）
- 盈利的原因需要确认是系统使然还是偶然

### ❌ 错误二：用结果倒推过程

赚了就认为自己是对的，亏了就认为系统有问题。

✅ **正确做法：** 过程比结果重要。执行到位的亏损交易也是好交易，执行不到位的盈利交易也有问题。

### ❌ 错误三：频繁修改系统

亏了一笔就改规则，赚了一笔又改回来。

✅ **正确做法：** 至少积累30笔交易数据再评估系统。单笔交易的盈亏不代表系统好坏。

### ❌ 错误四：复盘停留在纸面

写了很多分析但不去执行改进。

✅ **正确做法：** 每次复盘后列出1-3个具体可执行的改进点，在下一笔交易中刻意练习。

## 复盘工具建议

### 简单版：手机备忘录

每次交易记录在手机备忘录中：
```
日期 | 股票 | 买入价 | 卖出价 | 盈亏 | 理由 | 总结
```

### 进阶版：Excel表格

建立交易日志Excel，包含所有记录字段，自动计算胜率、盈亏比等指标。

### 专业版：交易日志APP

推荐： TradingView、富途牛牛的交易分析功能。

## 复盘检查清单

1. ✅ 每笔交易都有完整记录（买入理由、止损、目标）
2. ✅ 每周做一次周度复盘（30分钟）
3. ✅ 每月做一次月度复盘（1小时）
4. ✅ 分析盈利和亏损的共同点
5. ✅ 每次复盘后列出1-3个改进点
6. ✅ 改进点要在下一笔交易中执行
7. ✅ 评估系统是否需要调整（至少30笔数据）
