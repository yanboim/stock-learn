## 为什么复盘比交易更重要？

**交易让你赚当下的钱，复盘让你赚未来的钱。**

NBA球星科比每天训练后都会看录像回放，分析每个动作的得失。交易也一样——每笔交易结束后回头看看，才能知道：

- 你做对了什么（继续保持）
- 你做错了什么（需要改正）
- 你的系统哪里可以优化

:::tip[数据说话]
研究表明，坚持写交易日志并进行复盘的交易者，其长期收益率比不复盘的交易者高出**30-50%**。
原因很简单：复盘帮你避免了重复犯同样的错误。
:::

## 交易日志：复盘的基础

### 每笔交易必须记录的内容

**交易前（计划阶段）：**
- 日期和股票代码
- 买入理由（为什么选这只？）
- 入场信号（什么技术条件触发了买入？）
- 计划买入价位
- 止损价位
- 止盈目标价
- 计划仓位大小

**交易中（执行阶段）：**
- 实际买入价格
- 实际仓位
- 买入时的大盘环境
- 买入时的情绪状态（冷静/兴奋/焦虑）

**交易后（结果阶段）：**
- 卖出日期和价格
- 卖出理由（止损/止盈/其他）
- 实际盈亏金额和百分比
- 持仓天数

### 简易交易日志模板

```
日期：2025-06-05
股票：贵州茅台 600519
行业：白酒

【买入计划】
买入理由：季报超预期+北向资金持续流入+MACD金叉
入场信号：日线MACD金叉+放量突破MA20
计划买入价：1680元
止损价：1600元（-4.8%）
止盈目标：1850元（+10%）
仓位：20%（2万元）

【实际执行】
买入价：1685元（略高于计划）
仓位：20%
大盘环境：沪深300微涨0.3%
情绪：冷静

【卖出记录】
卖出日期：2025-06-25
卖出价：1820元
卖出理由：接近目标价+出现放量滞涨
实际盈亏：+135元/股（+8%）
持仓天数：20天

【复盘总结】
做对了：按计划执行，没有追高
可改进：可以等回调到MA20再入场
```

## 复盘的四个维度

### 维度一：交易逻辑复盘

问自己三个问题：

1. **买入逻辑是否成立？**
   - 当初买入的理由还在不在？
   - 如果让你重新选择，你还会买吗？

2. **逻辑是否被市场验证？**
   - 你预判的方向对了没有？
   - 你的论据是否正确？

3. **逻辑是否需要修正？**
   - 有哪些你没想到的因素？
   - 下次类似情况你会怎么调整？

### 维度二：执行纪律复盘

1. **你是否按计划执行了？**
   - 买入价格是否在计划范围内？
   - 止损是否严格执行？
   - 仓位是否按计划控制？

2. **是否有情绪化操作？**
   - 是否因为恐惧而提前卖出？
   - 是否因为贪婪而推迟止盈？
   - 是否因为焦虑而频繁交易？

### 维度三：技术分析复盘

1. **入场时机好不好？**
   - 回头看，有没有更好的买入点？
   - 入场信号是否准确？

2. **出场时机好不好？**
   - 卖出后股价怎么走的？
   - 是否卖早了或卖晚了？

3. **技术指标是否有效？**
   - 你使用的指标在这笔交易中表现如何？
   - 是否需要调整参数或更换指标？

### 维度四：资金管理复盘

1. **仓位是否合理？**
   - 仓位大小是否与信号强度匹配？
   - 是否过度集中或过度分散？

2. **风险控制是否到位？**
   - 单笔亏损是否在可承受范围内？
   - 总仓位的风险是否可控？

## 周度/月度总结

除了每笔交易的复盘，还要定期做**整体复盘**：

### 周度总结

1. 本周交易次数
2. 胜率（盈利笔数 ÷ 总笔数）
3. 平均盈利 vs 平均亏损（盈亏比）
4. 本周最大单笔盈亏
5. 本周情绪波动情况
6. 本周学到的教训

### 月度总结

1. 月度收益率
2. 最大回撤
3. 交易次数是否合理（过多=频繁交易，过少=过于犹豫）
4. 交易系统是否需要调整
5. 下个月的改进计划

## 复盘发现的常见问题模式

### 模式一：追高买入

**症状**：经常在股票大涨当天买入，第二天就开始跌

**原因**：看到涨了才追，FOMO（害怕错过）心理

**解决方案**：
- 制定买入等待清单，提前设好提醒价格
- 涨幅超过3%的当天不追，等回调

### 模式二：止损不执行

**症状**：到了止损位总想着"再等等"

**原因**：不甘心认亏，心存侥幸

**解决方案**：
- 用条件单自动止损
- 设定"到了止损位无脑执行"的铁律

### 模式三：频繁换股

**症状**：持仓不超过3天就换，手续费比利润还多

**原因**：缺乏耐心，总觉得自己手里的不如别人的

**解决方案**：
- 设定最短持仓周期（如至少持有5个交易日）
- 买入前明确预期持仓时间

### 模式四：盈利变亏损

**症状**：曾经浮盈20%以上，最后亏着出来

**原因**：不会移动止损，不会分批止盈

**解决方案**：
- 盈利超过10%后设置移动止损
- 到达目标价先卖一半，剩下的用跟踪止损

## 持续优化交易系统

复盘的最终目的是**优化你的交易系统**：

1. **统计系统表现**
   - 近20笔交易的胜率
   - 平均盈亏比
   - 最大连续亏损次数

2. **识别系统弱点**
   - 在什么市场环境下表现差？
   - 哪类股票容易出错？

3. **针对性改进**
   - 增加过滤条件（减少假信号）
   - 调整参数（提高适配性）
   - 补充规则（覆盖特殊情况）

:::tip[重要提醒]
优化系统时不要**过度拟合**——不要为了过去的数据完美而把系统调得太复杂。
好的系统是简单的、鲁棒的，在不同市场环境下都能稳定运行。
:::

## 实战检查清单

1. ✅ 每笔交易都写交易日志（即使是亏损的）
2. ✅ 每周做一次周度复盘
3. ✅ 每月做一次月度总结
4. ✅ 重点复盘亏损交易（亏损中学到的更多）
5. ✅ 根据复盘结果持续优化系统
