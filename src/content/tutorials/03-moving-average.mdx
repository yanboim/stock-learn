## 什么是均线？

均线（Moving Average，简称MA）就像一条**平滑的河流**，把杂乱无章的每日股价波动过滤掉噪音，展现出价格的总体趋势方向。

### 一个形象的比喻

想象你在看一条河的水流：
- 每一滴水的水流方向都不一样（就像每天的股价涨跌）
- 但如果你看整体水流的方向，就能判断河流往哪流（就像均线反映的趋势方向）

**均线 = 过去N天收盘价的平均值连成的线**

### 计算方法

以MA5（5日均线）为例：

```
MA5 = (今日收盘 + 昨日收盘 + 前日收盘 + ... + 5天前收盘) ÷ 5
```

每一天都重新计算，就形成了一条连续的曲线。

:::tip[核心原理]
均线是**滞后指标**——它跟着价格走，不会提前预测。但它的价值在于：
1. 过滤噪音，看清趋势
2. 提供支撑和压力位参考
3. 不同周期均线的交叉给出买卖信号
:::

## 常用均线周期及含义

| 均线 | 含义 | 用途 |
|---|---|---|
| **MA5** | 一周均价 | 超短线参考，价格紧贴MA5说明短期趋势强 |
| **MA10** | 两周均价 | 短线趋势参考 |
| **MA20** | 一个月均价 | **短线生命线**，跌破MA20短期走弱 |
| **MA60** | 一个季度均价 | **中线趋势线**，站上MA60中线看多 |
| **MA120** | 半年均价 | **长线牛熊分界线**，站上MA120长期看多 |
| **MA250** | 一年均价 | **年线**，牛熊市的终极分界 |

:::note[经验法则]
- 短线交易者关注：MA5、MA10、MA20
- 波段交易者关注：MA20、MA60、MA120
- 长线投资者关注：MA120、MA250
:::

## 均线的三大核心用法

### 用法一：判断趋势方向

**最简单的方法——看均线的方向：**

- 均线**向上** → 上升趋势 → 做多
- 均线**走平** → 震荡趋势 → 观望
- 均线**向下** → 下降趋势 → 不做多

**更可靠的方法——看价格与均线的位置关系：**

- 价格在均线**上方** → 多头趋势
- 价格在均线**下方** → 空头趋势

### 用法二：金叉与死叉

**金叉（买入信号）：** 短周期均线从下方穿越长周期均线

```
短期MA（如MA5）向上穿越长期MA（如MA20）
    \    /
     \  /  ← 金叉！看涨
      \/
```

**死叉（卖出信号）：** 短周期均线从上方穿越长周期均线

```
      /\
     /  \  ← 死叉！看跌
    /    \
短期MA向下穿越长期MA
```

**常用金叉/死叉组合：**

| 组合 | 信号类型 | 适用周期 |
|---|---|---|
| MA5 × MA20 | 短线信号 | 1-2周 |
| MA10 × MA30 | 短中线信号 | 2-4周 |
| MA20 × MA60 | 中线信号 | 1-3个月 |
| MA60 × MA120 | 长线信号 | 3-6个月 |

:::caution[注意]
金叉死叉在**趋势明确**时效果好，在**震荡市**中会频繁出现假信号！
一定要结合大趋势判断，不要盲目跟从每一个金叉。
:::

### 用法三：支撑与压力

均线天然具备**支撑**和**压力**的作用：

- **上升趋势中**：均线是支撑位，价格回调到均线附近容易反弹
  - MA20 支撑 → 短线回调买入点
  - MA60 支撑 → 中线回调买入点
- **下降趋势中**：均线是压力位，价格反弹到均线附近容易回落
  - MA20 压力 → 反弹卖出点
  - MA60 压励 → 中线反弹卖出点

## 经典均线策略

### 策略一：均线多头排列

当均线从上到下依次排列：**MA5 > MA10 > MA20 > MA60 > MA120**

这就像军队列阵一样整齐——所有周期都指向上涨，是**最强多头信号**。

### 策略二：均线空头排列

当均线从上到下依次排列：**MA120 > MA60 > MA20 > MA10 > MA5**

所有周期都指向下跌，是**最强空头信号**——不要逆势做多！

### 策略三：MA120 趋势过滤

这是最简单实用的策略：

- 价格站上MA120 → 只做多，不做空
- 价格跌破MA120 → 不做多，空仓观望
- 配合TD信号使用效果更佳

## 均线的局限性

1. **滞后性** — 均线跟随价格，信号出现时趋势可能已经走了一段
2. **震荡市假信号多** — 横盘时均线反复缠绕，金叉死叉频繁出错
3. **参数依赖** — 不同股票、不同周期可能需要不同参数
4. **不能单独使用** — 必须结合成交量、K线形态等一起判断

## 实战检查清单

1. ✅ 先看MA60/MA120判断大趋势方向
2. ✅ 大趋势向上时，等价格回到MA20附近考虑买入
3. ✅ 金叉信号出现后，结合成交量确认
4. ✅ 均线多头排列时持股信心更强
5. ✅ 跌破关键均线（如MA20）要及时减仓
