## 什么是趋势线？

趋势线就像高速公路的**车道线**——帮你看清价格运行的方向和边界。

画趋势线的方法很简单：
- **上涨趋势线** — 连接两个或以上的**低点**（支撑线）
- **下跌趋势线** — 连接两个或以上的**高点**（压力线）

### 趋势线的画法要点

1. **至少需要3个触点** — 2个点画线，第3个点确认
2. **触点越多越有效** — 触及趋势线后反弹的次数越多，这条线越重要
3. **不要强行画线** — 如果价格穿越趋势线太多次，说明这条线不成立

:::tip[关键原则]
趋势线被触及的次数越多、持续时间越长、跨越的时间周期越大，其有效性就越强。
一条持续2年的趋势线比一条只有2周的趋势线重要得多。
:::

## 三种基本趋势

### 上升趋势（多头市场）

特征：
- 价格不断创新高
- 回调低点不断抬高
- 均线多头排列
- 成交量上涨放量、回调缩量

操作策略：**顺势做多，回调到趋势线附近买入**

### 下降趋势（空头市场）

特征：
- 价格不断创新低
- 反弹高点不断降低
- 均线空头排列
- 成交量下跌放量、反弹缩量

操作策略：**不做多，空仓等待或找做空机会**

### 横盘趋势（震荡市场）

特征：
- 价格在一个区间内来回波动
- 高点和低点基本水平
- 均线缠绕

操作策略：**在支撑位买入、压力位卖出，或等待突破方向**

## 经典价格形态

### 🔼 反转形态 — 看涨

**1. 头肩底（Inverse Head and Shoulders）**

```
        右肩
       /    \
      /      \
左肩 /   头   \  ← 突破颈线 = 买入信号！
  \/  \/  \/
```

特征：三个低点，中间最低（头），两边差不多（肩）
含义：空方力量衰竭，多方即将反攻
目标：突破颈线后上涨幅度 ≈ 头部到颈线的距离

**2. 双底（W底）**

```
     /\
    /  \
   /    \
  /      \   ← 第二个底部不破前低 + 突破中间高点 = 买入
 \/        \/
```

特征：两次探底，第二次没有跌破第一次的低点
含义：底部支撑确认，多方蓄势待发

**3. 圆弧底**

```
    __
   /  \
  |    |
  |    |   ← 缓慢筑底，逐步放量 = 买入
   \__/
```

特征：价格缓慢下跌后缓慢回升，形成圆弧
含义：长期底部，主力缓慢吸筹

### 🔽 反转形态 — 看跌

**4. 头肩顶（Head and Shoulders）**

```
  左肩      右肩
    \      /
     \    /
      头       ← 跌破颈线 = 卖出信号！
     /    \
    /      \
```

特征：三个高点，中间最高（头），两边差不多（肩）
含义：多方力量衰竭，空方即将反攻
⚠️ 这是最可靠的反转形态之一

**5. 双顶（M头）**

```
  /\    /\
 /  \  /  \   ← 第二个顶部不破前高 + 跌破中间低点 = 卖出
/    \/    \
```

特征：两次冲高，第二次没有突破第一次的高点
含义：顶部压力确认，多方进攻失败

### ▶️ 中继形态 — 趋势继续

**6. 上升三角形**

```
  ------  ← 水平压力线
  /\  /\  /
 /  \/  \/    ← 突破水平压力 = 继续上涨
```

特征：水平压力线 + 不断抬高的低点
含义：买方越来越强，一旦突破将加速上涨

**7. 下降三角形**

```
 \  /\  /\
  \/  \/  \   ← 跌破水平支撑 = 继续下跌
  ------  ← 水平支撑线
```

特征：水平支撑线 + 不断降低的高点
含义：卖方越来越强，一旦跌破将加速下跌

**8. 旗形**

```
平行通道
 /|  /|
/ | / |   ← 与主趋势方向相反的短暂整理
  |/  |/
  /   /
 /   /    ← 突破通道 = 趋势继续
```

特征：价格在两条平行线内小幅整理
含义：趋势"中场休息"，之后继续原方向运行

## 形态分析的实战要点

### 要点一：等待确认，不要提前操作

- 双底形态完成 = 价格突破两个底部之间的反弹高点
- 头肩底形态完成 = 价格突破颈线
- **形态完成前一切都是猜测**

### 要点二：成交量确认

- 突破时必须放量（否则可能是假突破）
- 形态构建过程中缩量（说明多空双方犹豫）
- 放量突破 = 真突破的概率大大增加

### 要点三：测量目标价

很多形态完成后，可以估算目标价：

- **头肩形态**：目标 = 颈线 ± (头部到颈线的距离)
- **双顶/双底**：目标 = 突破点 ± (顶部/底部到颈线的距离)
- **三角形**：目标 = 突破点 ± (三角形最宽处的距离)

:::caution[注意]
目标价只是参考，不是绝对的。到了目标价附近要结合其他指标判断是否出场。
:::

### 要点四：形态失败

不是所有形态都会按预期走。如果：
- 突破后又跌回形态内部 → **形态失败** → 可能走出相反方向
- 要设好止损，形态失败时及时离场

## 常见错误

### ❌ 在小周期上找形态

5分钟线上的"头肩顶"几乎没有意义。

✅ 至少在日线以上周期找形态，周线形态更可靠。

### ❌ 强行画趋势线

不是每段走势都有清晰的形态，不要为了画而画。

✅ 有明确形态就分析，没有就看其他指标。

### ❌ 忽略大趋势

下跌趋势中的"双底"可能只是下跌中继。

✅ 先判断大趋势方向，再在小级别找形态。

## 实战检查清单

1. ✅ 先判断大趋势方向（周线/月线）
2. ✅ 在日线以上周期识别形态
3. ✅ 等形态完成（突破确认）再操作
4. ✅ 突破必须有成交量配合
5. ✅ 设好止损，形态失败及时离场
