## 什么是趋势？

趋势就是价格的运行方向。识别趋势是技术分析的基石。

**道氏理论定义的三种趋势：**

| 趋势类型 | 持续时间 | 特征 |
|---|---|---|
| **主要趋势** | 数月至数年 | 大方向（牛/熊） |
| **次要趋势** | 数周至数月 | 主要趋势中的回调/反弹 |
| **短期趋势** | 数天至数周 | 日内波动 |

**核心原则：顺势交易。不要和趋势作对。**

## 趋势线的画法

### 上升趋势线

**画法：** 连接两个或以上的**低点**，画一条向上倾斜的直线。

**有效条件：**
1. 至少需要两个低点来画线
2. 三个以上触点更可靠
3. 触点越多、时间跨度越大越有效
4. 趋势线不能太陡（45度左右最理想）

**含义：**
- 价格在趋势线以上 = 上升趋势完好
- 价格回调到趋势线附近 = 可能是买入机会
- 价格跌破趋势线 = 上升趋势可能结束

### 下降趋势线

**画法：** 连接两个或以上的**高点**，画一条向下倾斜的直线。

**含义：**
- 价格在趋势线以下 = 下降趋势完好
- 价格反弹到趋势线附近 = 可能是卖出/减仓机会
- 价格突破趋势线 = 下降趋势可能结束

### 趋势线的有效性和失效

**趋势线越有效，说明它越被市场认可。**

| 特征 | 有效性 |
|---|---|
| 触及3次以上 | 非常有效 |
| 时间跨度数月以上 | 非常有效 |
| 接近45度角 | 最有效 |
| 太陡（&gt;60度） | 容易被突破 |
| 太平（&lt;20度） | 趋势不明显 |

**趋势线失效的判断：**
- 日内跌破但收盘收回来 = 假跌破，趋势线仍有效
- 收盘价跌破趋势线 = 趋势线可能失效
- 连续3天收盘在趋势线以下 = 趋势线确认失效

:::tip[趋势线的正确用法]
趋势线不是精确工具，而是一个**区域**。不要执着于精确到分的画线，而是看趋势线附近的反应。
:::

## 经典价格形态

### 反转形态（趋势即将改变）

#### 头肩顶（看跌反转）⭐⭐⭐

**形态描述：**
```
        左肩       头       右肩
         /\       /\       /\
        /  \     /  \     /  \
       /    \   /    \   /    \
──────/──────\─/──────\─/──────\────── 颈线
```

**三个阶段：**
1. **左肩**：价格上涨到高点后回落
2. **头部**：价格再创新高后回落（成交量往往减少）
3. **右肩**：价格反弹但未超过头部高度

**确认条件：** 价格跌破颈线 = 头肩顶确认

**目标跌幅：** 从颈线往下，约等于头部到颈线的距离

**实战要点：**
- 头部新高但成交量减少 = 顶背离预警
- 右肩成交量通常最小
- 跌破颈线时往往放量

#### 头肩底（看涨反转）⭐⭐⭐

头肩顶的镜像，出现在下跌趋势末端：
1. 左肩：下跌后反弹
2. 头部：再创新低后反弹
3. 右肩：再次下跌但未创新低
4. 突破颈线 = 确认反转

**可靠性：** 头肩底比头肩顶更可靠（底部形态通常更扎实）

#### 双底（W底）⭐⭐⭐

**形态描述：**
```
    /\     /\
   /  \   /  \
──/────\_/────\── 颈线
```

**特征：**
- 两个低点价格接近
- 第二个低点可能略高于第一个
- 突破颈线确认

**目标涨幅：** 颈线往上，约等于颈线到低点的距离

#### 双顶（M顶）⭐⭐⭐

双底的镜像，出现在上涨趋势末端：
- 两个高点价格接近
- 第二个高点可能略低于第一个
- 跌破颈线确认

### 继续形态（趋势暂时休息）

#### 三角形

**上升三角形（看涨）：**
```
──────────── 水平阻力线
\          /
 \        /
  \      /
   \    /
    \  /
     \/  ← 逐渐抬高的低点
```
- 水平压力线 + 上升支撑线
- 突破水平线 = 看涨信号

**下降三角形（看跌）：**
- 水平支撑线 + 下降压力线
- 跌破水平线 = 看跌信号

**对称三角形：**
- 两条收敛的趋势线
- 方向不确定，等突破确认
- 突破方向通常延续之前的趋势

#### 旗形

**特征：**
- 价格急涨/急跌后进入短期整理
- 整理区间与主趋势方向相反的平行通道
- 突破通道后通常延续原趋势

**可靠性：** ⭐⭐⭐⭐（旗形是趋势继续的信号）

### 形态的交易策略

**策略一：突破入场**
- 等待价格突破颈线/趋势线
- 突破当天成交量放大 = 确认有效
- 在突破价附近买入

**策略二：回踩入场**
- 突破后等价格回踩到颈线
- 颈线从压力变为支撑
- 回踩不破颈线时买入

**策略三：形态失败**
- 如果形态没有按预期突破
- 反而向反方向运动 = 形态失败
- 形态失败后的运动往往很剧烈

## 支撑与压力

### 什么是支撑和压力？

**支撑位：** 价格跌到这个位置容易止跌反弹（买方力量集中的区域）
**压力位：** 价格涨到这个位置容易受阻回落（卖方力量集中的区域）

### 常见的支撑/压力位

| 类型 | 说明 | 可靠性 |
|---|---|---|
| **前期高点/低点** | 历史上重要的转折点 | ⭐⭐⭐⭐⭐ |
| **整数关口** | 如10元、20元、50元、100元 | ⭐⭐⭐ |
| **均线** | MA20、MA60等重要均线 | ⭐⭐⭐⭐ |
| **跳空缺口** | 缺口的上沿或下沿 | ⭐⭐⭐⭐ |
| **密集成交区** | 之前大量交易的价位 | ⭐⭐⭐⭐ |
| **趋势线** | 上升趋势线或下降趋势线 | ⭐⭐⭐⭐ |

### 支撑压力的转换

**一个重要规律：** 一旦支撑被有效跌破，它就变成了压力。一旦压力被有效突破，它就变成了支撑。

```
支撑被跌破 → 变成新的压力位
压力被突破 → 变成新的支撑位
```

## 趋势分析检查清单

1. ✅ 先判断大趋势方向（日线+周线）
2. ✅ 画趋势线验证趋势（至少两个触点）
3. ✅ 识别关键支撑和压力位
4. ✅ 观察是否出现反转形态或继续形态
5. ✅ 突破需要成交量确认
6. ✅ 假突破要设好止损
7. ✅ 顺势交易，不要逆势抄底
