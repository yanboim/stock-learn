## 为什么仓位管理比选股更重要？

先做一个思想实验：

你有两只股票各50%概率赚钱：
- 股票A：赚50%或亏50%
- 股票B：赚20%或亏10%

如果你把所有钱押在股票A上：
- 赢了：100万变150万
- 输了：100万变50万

但如果你只用20%仓位买股票A，80%买股票B呢？
- A赢了B赢了：108万（+8%）
- A赢了B输了：102万（+2%）
- A输了B赢了：88万（-12%）
- A输了B输了：82万（-18%）

你看，通过仓位管理，最差情况从亏50%变成了亏18%。

:::tip[核心真理]
**选股决定了你能不能赚钱，仓位管理决定了你能活多久。**

一个好的仓位管理系统可以让你在选错股票时少亏，在选对股票时多赚。
:::

## 仓位管理的核心原则

### 原则一：永远不满仓

| 总仓位 | 市场状态 | 含义 |
|---|---|---|
| **100%** | 极度看多 | 几乎不应该出现 |
| **70-80%** | 看多 | 强势行情 |
| **50-60%** | 中性 | 正常市场 |
| **30-40%** | 谨慎 | 震荡市 |
| **10-20%** | 看空 | 弱势行情 |
| **0%** | 极度看空 | 空仓观望 |

### 原则二：单只股票仓位上限

| 资金规模 | 单只上限 | 建议持仓数量 |
|---|---|---|
| 10万以下 | 20% | 3-5只 |
| 10-50万 | 15% | 5-8只 |
| 50-100万 | 10% | 8-12只 |
| 100万以上 | 5-8% | 10-15只 |

### 原则三：单笔最大亏损

**铁律：单笔亏损不超过总资金的2%**

```
单笔最大亏损 = 总资金 × 2%

10万资金 → 单笔最多亏2000元
50万资金 → 单笔最多亏10000元
100万资金 → 单笔最多亏20000元
```

## 仓位管理实战方法

### 方法一：等权分配法

**最简单的仓位管理方法。**

把资金平均分配到N只股票上：
- 10万资金，买5只股票 → 每只2万

**优点：** 简单，分散风险
**缺点：** 不考虑不同股票的风险差异

### 方法二：风险平价法

**根据风险大小分配仓位。**

```
核心思路：每只股票的风险贡献相同

波动大的股票 → 小仓位
波动小的股票 → 大仓位
```

**举例：**
- 股票A：日波动率2% → 仓位15%
- 股票B：日波动率1% → 仓位30%
- 股票C：日波动率3% → 仓位10%

三只股票的风险贡献大致相同（2%×15% ≈ 1%×30% ≈ 3%×10%）

### 方法三：凯利公式

**数学上最优的仓位计算方法。**

```
最优仓位比例 = (胜率×平均盈利比 - 败率×平均亏损比) / 平均盈利比

简化版：
f = (p × b - q) / b

p = 胜率
q = 败率 = 1 - p
b = 盈亏比（平均盈利/平均亏损）
```

**举例：**
```
胜率 = 50%（p = 0.5）
平均盈利 = 15%
平均亏损 = 5%
盈亏比 = 3:1（b = 3）

凯利仓位 = (0.5 × 3 - 0.5) / 3 = 1.0 / 3 = 33.3%
```

:::caution[凯利公式的风险]
凯利公式给出的是数学上最优的仓位，但实际使用中应该用 **半凯利**（即凯利值的一半），因为：
1. 你的胜率估算可能不准确
2. 凯利值的波动极大，心理上很难承受
3. 半凯利能在保持大部分收益的同时大幅降低波动
:::

### 方法四：分批建仓法

**不一次性买入，分多次建仓。**

**金字塔加仓法：**

```
第一次买入：50%仓位（底仓）
确认趋势后加仓：30%
趋势加速再加：20%

（越买越少，成本在底部）
```

**倒金字塔加仓（不推荐）：**
```
第一次买入：20%
加仓：30%
再加仓：50%

（越买越多，成本在顶部，风险极大）
```

**为什么金字塔加仓更好？**

```
例子：股价从25元涨到30元

金字塔加仓：
25元买50% = 均价25
28元买30% = 均价26.2
30元买20% = 均价27.0
综合均价约 26.0元

倒金字塔加仓：
25元买20% = 均价25
28元买30% = 均价27.0
30元买50% = 均价29.2
综合均价约 29.2元
```

金字塔加仓的均价远低于倒金字塔。

### 方法五：动态调仓法

**根据市场状态动态调整总仓位。**

| 市场信号 | 操作 |
|---|---|
| 均线多头排列 + MACD零轴上方 | 总仓位70-80% |
| 均线走平 + MACD在零轴附近 | 总仓位50% |
| 均线空头排列 + MACD零轴下方 | 总仓位20-30% |
| 出现恐慌性大跌 | 总仓位降到10%以下 |

## 分散投资的艺术

### 行业分散

不要把所有钱都买同一行业的股票：

| 分散程度 | 示例 | 评价 |
|---|---|---|
| **同行业3只** | 茅台+五粮液+泸州老窖 | ❌ 不算分散 |
| **2-3个行业** | 消费+科技+银行 | ✅ 基本分散 |
| **4-5个行业** | 消费+科技+医药+新能源+金融 | ✅ 充分分散 |
| **超过8个行业** | 分散到太多行业 | ⚠️ 过度分散 |

### 风格分散

同时配置不同风格的股票：
- 价值股（低PE、高股息）
- 成长股（高增长、高估值）
- 防御股（波动小、稳定）

### 时间分散

不要在同一天把所有钱买完：
- 分3-5个交易日建仓
- 每次建仓间隔1-2天
- 如果市场大跌，可以趁低价多买一点

## 复利的力量

**看看这个对比：**

| 场景 | 月收益率 | 1年后 | 3年后 | 5年后 |
|---|---|---|---|---|
| **保守** | 1% | 11.3万 | 13.6万 | 16.3万 |
| **稳健** | 3% | 14.3万 | 23.0万 | 37.0万 |
| **激进** | 5% | 17.9万 | 38.2万 | 81.4万 |

初始资金10万，每月追加5000元。注意：激进收益意味着更大的风险。

:::caution[复利的另一面]
复利在亏损时同样可怕：

亏损5% → 需要赚5.3%回本
每月亏2%，一年后资金缩水21.5%
每月亏5%，一年后资金缩水45.8%

这就是为什么保护本金比追求收益更重要。
:::

## 仓位管理检查清单

1. ✅ 永远不满仓，保留至少20%现金
2. ✅ 单只股票不超过总资金的15-20%
3. ✅ 单笔亏损不超过总资金的2%
4. ✅ 分散到至少3-5个不同行业
5. ✅ 用金字塔法分批建仓
6. ✅ 根据市场状态动态调整总仓位
7. ✅ 每笔交易前计算风险收益比，至少2:1
