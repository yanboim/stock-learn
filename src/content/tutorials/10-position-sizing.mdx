## 为什么仓位管理比选股更重要？

一个残酷的真相：**即使你的胜率高达70%，如果每次满仓进出，一次大亏就能让你回到起点。**

看两个对比：

**投资者A** — 选股能力很强，胜率70%
- 每次满仓买入一只股票
- 赚的时候赚20%，亏的时候亏30%
- 10次交易：7次赚 = +140%，3次亏 = -90%
- 但问题是：如果在第3次满仓亏损70%，后面的7次赢利都救不回来

**投资者B** — 选股能力一般，胜率50%
- 每次只用20%仓位
- 严格止损5%，止盈15%
- 10次交易：5次赚 = 5×3% = +15%，5次亏 = 5×1% = -5%
- **净赚10%，而且没有大起大落**

:::tip[核心结论]
**仓位管理不能让你买对股票，但能让你在买错的时候活下来，在买对的时候赚到更多。**
:::

## 仓位管理的四大原则

### 原则一：永远不满仓

| 市场环境 | 建议最大仓位 | 保留现金 |
|---|---|---|
| 强势牛市 | 80-90% | 10-20% |
| 温和上涨 | 60-80% | 20-40% |
| 震荡盘整 | 40-60% | 40-60% |
| 弱势下跌 | 20-40% | 60-80% |
| 极端熊市 | 0-20% | 80-100% |

保留现金的目的：
1. 应对意外大跌（有子弹补仓）
2. 抓住突然出现的机会
3. 心理安慰（不至于恐慌割肉）

### 原则二：单只股票仓位上限

- **单只股票不超过总资金的20-25%**
- 同时持有 **4-6只** 股票为宜
- 最少持有3只（分散风险）
- 最多不超过8只（精力不够管不过来）

### 原则三：分批建仓

不要一次性全仓买入，分2-3批建仓：

**三步建仓法：**
1. **底仓**（33%）— 信号初步出现，试探性入场
2. **加仓**（33%）— 信号确认，趋势进一步明朗
3. **追仓**（33%）— 突破确认，趋势完全确立

:::tip[关键]
**只有在底仓盈利的情况下才能加仓！**
如果底仓已经亏损，不要"拉低成本"加仓——这会让你在错误的股票上越陷越深。
:::

### 原则四：不同股票之间的相关性

分散投资不等于"随便买几只"。

- ❌ 买5只银行股 → 不是分散，是一起涨一起跌
- ❌ 买3只新能源股 → 同行业同涨同跌
- ✅ 买1只银行 + 1只消费 + 1只科技 + 1只医药 → 真正的分散

**目标：持仓之间的相关性越低越好。**

## 仓位计算方法

### 方法一：固定风险法（推荐新手）

根据你的**单笔最大可承受亏损**来计算仓位：

```
买入仓位 = 单笔最大亏损金额 ÷ 止损幅度
```

**例子：**
- 总资金：10万元
- 单笔最大亏损：2%（2000元）
- 止损幅度：8%

买入金额 = 2000 ÷ 8% = **25000元**（25%仓位）

### 方法二：凯利公式（进阶）

凯利公式帮你计算**数学上最优的仓位比例**：

```
最优仓位比例 = (胜率 × 盈亏比 - 败率) / 盈亏比
```

**例子：**
- 你的胜率：50%
- 平均盈利：平均亏损 = 2:1（盈亏比2）
- 最优比例 = (0.5×2 - 0.5) / 2 = **25%**

:::caution[实战建议]
实际操作中，建议用**凯利值的一半**（半凯利），因为：
1. 估算的胜率和盈亏比不一定准确
2. 半凯利更保守，回撤更小
3. 心理上更容易承受波动
:::

### 方法三：等权分配法（最简单）

每只股票分配**相同比例**的资金：

- 买4只 → 每只25%
- 买5只 → 每只20%
- 买6只 → 每只16-17%

简单粗暴但有效，适合新手。

## 动态仓位调整

仓位不是一成不变的，应该根据市场变化动态调整：

### 根据市场强弱调仓

- 市场走强 → 逐步加仓（不要一步到位）
- 市场走弱 → 逐步减仓（不要死扛）
- 信号不明确 → 维持当前仓位不动

### 根据持仓盈亏调仓

- 盈利的股票 → 可以加仓，让利润奔跑
- 亏损的股票 → 不要加仓，严格止损
- 盈利大幅扩大 → 用移动止损保护利润

### 根据估值水平调仓

- 整体估值低（如沪深300 PE < 10）→ 可以加大仓位
- 整体估值高（如沪深300 PE > 15）→ 应当降低仓位

## 常见仓位管理错误

### ❌ 金字塔加仓方向搞反

正确：底部买得多，越往上加得越少（正金字塔）
错误：底部买得少，越往上加得越多（倒金字塔）→ 高位重仓，一回调就大亏

### ❌ 亏损加仓"拉低成本"

买了跌了 → 加仓 → 继续跌 → 继续加 → 越套越深

✅ 只有盈利的股票才加仓，亏损的股票到止损位就走人。

### ❌ 频繁满仓空仓

今天满仓明天空仓，后天又满仓 → 手续费大增，心态崩坏

✅ 渐进式调整仓位，每次调整10-20%。

## 实战检查清单

1. ✅ 买入前确定仓位大小（不超过总资金25%）
2. ✅ 分2-3批建仓，底仓盈利后再加
3. ✅ 持仓分散在不同行业（相关性低）
4. ✅ 总仓位根据市场环境调整
5. ✅ 永远保留至少20%现金
6. ✅ 单笔最大亏损不超过总资金的2-3%
