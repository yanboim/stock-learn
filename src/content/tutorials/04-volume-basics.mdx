## 为什么成交量这么重要？

如果K线是市场的**语言**，那么成交量就是市场的**测谎仪**。

股价可以"骗人"——主力资金可以操纵短期价格。但**成交量很难造假**，因为每一手交易都是真金白银。

:::tip[华尔街名言]
**"量在价先"** — 成交量的变化往往领先于价格变化。
价格上涨但没有成交量配合，就像没有油量的汽车，跑不远。
:::

## 量的基本概念

### 什么是成交量？

- 成交量 = 一定时间内买卖双方成交的**股票数量**
- 成交额 = 成交量 × 成交价格（反映资金规模）
- 在K线图下方通常用**柱状图**表示

### 放量 vs 缩量

| 状态 | 含义 | 形象理解 |
|---|---|---|
| **放量** | 成交量明显大于近期平均 | 像高速公路突然车流暴增——有大事发生 |
| **缩量** | 成交量明显小于近期平均 | 像商场里没几个顾客——大家都在观望 |
| **天量** | 创出阶段性甚至历史最大量 | 像洪水爆发——极端情绪的释放 |
| **地量** | 创出阶段性甚至历史最小量 | 像干涸的河床——交易极度低迷 |

## 量价关系的8种经典组合

### 🟢 看涨组合

**1. 量增价涨（健康上涨）**
- 成交量放大 + 价格上涨
- 含义：买方积极入场，上涨有真实资金支撑
- 就像涨潮，水位升高且水量充沛

**2. 量缩价稳（缩量企稳）**
- 成交量萎缩 + 价格不再下跌
- 含义：卖方力量衰竭，下跌动能耗尽
- 在底部出现 → 可能即将反弹

**3. 量增价平（底部放量）**
- 成交量放大 + 价格横盘
- 含义：有大量资金在"偷偷"买入
- 在底部出现 → ⚠️ 可能是主力建仓信号

**4. 量缩价涨（锁筹上涨）**
- 成交量减少 + 价格继续上涨
- 含义：持有者不愿卖出，筹码锁定
- 在上涨中段出现 → 涨势还能持续

### 🔴 看跌组合

**5. 量增价跌（放量下跌）**
- 成交量放大 + 价格下跌
- 含义：大量卖盘涌出，买方接不住
- 就像大坝决堤——恐慌性抛售

**6. 量增价滞（放量滞涨）**
- 成交量放大 + 价格涨不动
- 含义：有大量资金想卖，但买方也在接
- 在高位出现 → ⚠️ 可能是主力出货

**7. 量缩价跌（无量阴跌）**
- 成交量萎缩 + 价格持续下跌
- 含义：没人愿意买，市场冷清
- 就像慢慢失血——最可怕的下跌方式

**8. 天量天价（见顶信号）**
- 创出天量 + 价格创新高后回落
- 含义：所有人都冲进来了，没有新买盘
- ⚠️ 强烈的见顶信号

## 成交量的实战应用

### 应用一：确认突破的有效性

价格突破关键位置时（如前高、均线、趋势线）：
- ✅ **放量突破** → 突破有效，可以跟进
- ❌ **缩量突破** → 假突破概率大，谨慎

### 应用二：判断趋势强弱

- 上涨趋势中：放量上涨 + 缩量回调 = 健康上涨
- 上涨趋势中：缩量上涨 + 放量下跌 = 趋势可能结束

### 应用三：识别主力动向

- 底部放量横盘 → 主力可能在**建仓**
- 上涨中放量突破 → 主力可能在**拉升**
- 高位放量不涨 → 主力可能在**出货**
- 下跌中放量暴跌 → 主力可能在**砸盘**

### 应用四：判断底部区域

- 经过长时间下跌后出现**地量** → 卖方力量衰竭
- 地量之后出现**放量阳线** → 底部确认信号

## 成交量的3个陷阱

### 陷阱一：低位放量不一定涨

底部放量可能是：
- 主力建仓（看涨）
- 最后一波恐慌性抛售（还可能继续跌）

✅ 等底部放量后价格企稳确认再入场

### 陷阱二：高位放量不一定跌

高位放量可能是：
- 主力出货（看跌）
- 新资金大量涌入继续推升（可能继续涨）

✅ 结合K线形态和后续走势确认

### 陷阱三：量价背离要警惕

- 价格创新高但成交量萎缩 → 上涨动力不足
- 价格创新低但成交量萎缩 → 下跌动力不足

## 实战检查清单

1. ✅ 每次看K线一定要同时看下方的成交量
2. ✅ 上涨必须放量，回调允许缩量
3. ✅ 突破关键位必须有成交量配合
4. ✅ 高位放量滞涨要警惕
5. ✅ 地量见地价，天量见天价
