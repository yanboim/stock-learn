## 为什么成交量比价格更重要？

股市中有一句老话：**"量在价先"**。

如果说价格是市场的"嘴巴"（说出大家想听的话），那成交量就是市场的"钱包"（真金白银投了票）。价格可以骗人，但成交量很难造假。

:::tip[一句话理解]
成交量 = 资金的脚印。价格上涨 + 成交量放大 = 真突破。价格上涨 + 成交量萎缩 = 假突破。
:::

## 成交量的基础概念

### 什么是成交量？

- **成交量（Volume）**：一段时间内买卖双方成交的股票数量
- **成交额（Turnover）**：成交量 × 成交价格 = 实际交易金额

**为什么要看成交额？**

同样是100万成交量，10元的股票（成交额1000万）和100元的股票（成交额1亿），资金规模完全不同。成交额更能反映真实的资金参与程度。

### 成交量的基本单位

| 术语 | 含义 |
|---|---|
| **放量** | 成交量明显大于近期平均水平（1.5倍以上） |
| **缩量** | 成交量明显小于近期平均水平（0.5倍以下） |
| **天量** | 成交量创出很长时间内的新高 |
| **地量** | 成交量创出很长时间内的新低 |
| **堆量** | 成交量持续放大，像堆积起来一样 |

## 量价关系的8种经典组合

### 看涨组合（4种）

**1. 放量上涨（量增价涨）⭐**
- 含义：买方积极入场，资金推动价格上涨
- 可靠性：**最高**
- 操作：持股或买入

**2. 缩量回调（量缩价跌）**
- 含义：上涨趋势中的正常调整，卖方不积极
- 可靠性：较高
- 操作：回调到支撑位时可以加仓

**3. 底部放量（地量后放量）**
- 含义：长期低迷后，资金开始入场
- 可靠性：较高（需要确认）
- 操作：关注是否形成底部，准备买入

**4. 突破放量（放量突破阻力位）⭐**
- 含义：资金大力突破关键位置，意愿强烈
- 可靠性：**很高**
- 操作：突破确认后跟进

### 看跌组合（4种）

**5. 放量下跌（量增价跌）⭐**
- 含义：卖方积极抛售，资金出逃
- 可靠性：**很高**
- 操作：远离或止损

**6. 缩量上涨（量缩价涨）**
- 含义：价格上涨但资金参与不足，涨势不牢靠
- 可靠性：较高
- 操作：警惕假突破，不宜追高

**7. 天量天价（天量后滞涨）⭐**
- 含义：天量出现但价格涨不动，主力可能在出货
- 可靠性：**很高**
- 操作：考虑减仓或清仓

**8. 顶部放量滞涨**
- 含义：成交量巨大但价格不涨，多空分歧严重
- 可靠性：较高
- 操作：提高警惕，设置止损

## 量价分析的核心原则

### 原则一：量价必须配合

**健康的上涨 = 价格上涨 + 成交量放大**
**不健康的上涨 = 价格上涨 + 成交量萎缩**

如果一个趋势没有成交量的支持，这个趋势是不可持续的。

### 原则二：关注异常成交量

当成交量突然放大到正常水平的3-5倍以上时，往往意味着有大事发生：
- 有利好消息刺激
- 主力资金大举买入或卖出
- 关键位置突破或破位

### 原则三：地量见地价

当成交量萎缩到极致（地量）时，往往意味着：
- 卖方已经卖完了
- 观望情绪浓厚
- 底部可能不远了

**但注意：** 地量之后可能还有地量，不能仅凭地量就买入。需要等放量确认。

## 主力资金动向识别

### 主力吸筹（买入）特征

1. **长期缩量横盘后突然放量**
2. **放量阳线多于放量阴线**
3. **下跌时缩量，上涨时放量**
4. **盘中急跌但尾盘回升**

### 主力出货（卖出）特征

1. **天量出现但价格涨不动**
2. **放量阴线增多**
3. **上涨缩量，下跌放量**
4. **高开后低走，放量收阴**

### 主力洗盘特征

1. **缩量急跌后快速回升**
2. **跌破重要均线但很快收回来**
3. **成交量先大后小（洗出不坚定的人）**
4. **下跌幅度有限（通常不超过10-15%）**

:::caution[注意]
主力行为分析需要经验积累，以上特征仅供参考。不要仅凭一两个特征就断定主力意图，必须综合判断。
:::

## 量价分析实战案例

### 模拟案例一：放量突破确认

某消费股在25元附近横盘整理了一个月，日均成交量约50万股。

某天：
- 收盘价突破到26元（放量突破25元阻力位）
- 成交量放大到150万股（3倍于日常）
- MACD金叉确认

**解读：** 放量突破 + 指标共振，这是一次有效的突破。可以在26元附近买入。

**止损：** 跌回25元以下（突破失败）
**目标：** 28-30元

### 模拟案例二：缩量上涨警示

某科技股从30元涨到35元，但成交量从日均100万股萎缩到40万股。

**解读：** 价格创新高但成交量持续萎缩 = 量价背离。上涨缺乏资金支持，随时可能回调。

**操作：** 已经持有的可以考虑减仓50%。未持有的不要追高。

## 量价分析检查清单

1. ✅ 上涨必须放量，缩量上涨不可靠
2. ✅ 下跌缩量比下跌放量好（说明不是恐慌抛售）
3. ✅ 突破关键位置需要放量确认
4. ✅ 天量天价要警惕，地量地价是机会
5. ✅ 量价配合比单一指标更可靠
6. ✅ 关注成交量的相对变化，不看绝对值
7. ✅ 量在价先——成交量往往领先价格变化
